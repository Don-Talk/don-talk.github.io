## 📖 学习步骤

### 1. 环境准备

#### 必需软件

- **JDK 17**（必须，Spring AI不支持JDK 8）
- **Maven 3.6+** 或 Gradle 7.6+
- **IDE**：IntelliJ IDEA（推荐）或 VS Code

#### 验证安装

打开命令行，执行以下命令：

```bash
# 检查Java版本（应显示17或更高）
java -version

# 检查Maven版本
mvn -version
```

如果显示版本号，说明安装成功。

#### IDE推荐

**IntelliJ IDEA**（社区版即可）
- 下载地址：https://www.jetbrains.com/idea/download/
- 优势：对Spring Boot支持最好，智能提示强大

### 2. 创建Spring Boot项目

#### 方式一：使用Spring Initializr（推荐）

1. 访问 https://start.spring.io/
2. 配置项目信息：
   - **Project**: Maven
   - **Language**: Java
   - **Spring Boot**: 3.2.x（最新稳定版）
   - **Group**: com.example
   - **Artifact**: ai-demo
   - **Package name**: com.example.aidemo
   - **Packaging**: Jar
   - **Java**: 17

3. 添加依赖：
   - Spring Web
   - Spring AI OpenAI（后续手动修改为DeepSeek）

4. 点击 **GENERATE** 下载项目
5. 解压后用IDEA打开

#### 方式二：手动创建Maven项目

创建一个空的Maven项目，然后在 `pom.xml` 中添加以下依赖：

```xml
<?xml version="1.0" encoding="UTF-8"?>
<project xmlns="http://maven.apache.org/POM/4.0.0"
         xmlns:xsi="http://www.w3.org/2001/XMLSchema-instance"
         xsi:schemaLocation="http://maven.apache.org/POM/4.0.0 
         http://maven.apache.org/xsd/maven-4.0.0.xsd">
    <modelVersion>4.0.0</modelVersion>

    <parent>
        <groupId>org.springframework.boot</groupId>
        <artifactId>spring-boot-starter-parent</artifactId>
        <version>3.2.5</version>
        <relativePath/>
    </parent>

    <groupId>com.example</groupId>
    <artifactId>ai-demo</artifactId>
    <version>1.0-SNAPSHOT</version>

    <properties>
        <java.version>17</java.version>
        <spring-ai.version>1.0.0-M5</spring-ai.version>
    </properties>

    <dependencies>
        <!-- Spring Boot Web -->
        <dependency>
            <groupId>org.springframework.boot</groupId>
            <artifactId>spring-boot-starter-web</artifactId>
        </dependency>

        <!-- Spring AI OpenAI Starter -->
        <dependency>
            <groupId>org.springframework.ai</groupId>
            <artifactId>spring-ai-openai-spring-boot-starter</artifactId>
        </dependency>
    </dependencies>

    <!-- Spring AI BOM -->
    <dependencyManagement>
        <dependencies>
            <dependency>
                <groupId>org.springframework.ai</groupId>
                <artifactId>spring-ai-bom</artifactId>
                <version>${spring-ai.version}</version>
                <type>pom</type>
                <scope>import</scope>
            </dependency>
        </dependencies>
    </dependencyManagement>

    <!-- Spring Milestones仓库（下载Spring AI需要） -->
    <repositories>
        <repository>
            <id>spring-milestones</id>
            <name>Spring Milestones</name>
            <url>https://repo.spring.io/milestone</url>
            <snapshots>
                <enabled>false</enabled>
            </snapshots>
        </repository>
    </repositories>
</project>
```

### 3. 获取DeepSeek API Key

#### 步骤1：注册账号

1. 访问 DeepSeek 官网：https://www.deepseek.com/
2. 点击右上角 **API开放平台**
3. 使用手机号或邮箱注册账号

#### 步骤2：创建API Key

1. 登录后进入API开放平台控制台
2. 找到 **API Keys** 菜单
3. 点击 **创建新的API Key**
4. 复制生成的Key（格式：`sk-xxxxxxxxxxxxxxxx`）

**⚠️ 重要**：API Key只显示一次，请立即保存！

#### 步骤3：充值（可选）

- DeepSeek提供少量免费额度供测试
- 如需大量使用，需要充值（价格非常便宜）
- 新手学习阶段，免费额度足够

### 4. 编写第一个AI接口

#### 配置文件

在 `src/main/resources/application.properties` 中添加：

```properties
# 服务端口
server.port=8080

# DeepSeek API配置
spring.ai.openai.api-key=sk-你的API-Key
spring.ai.openai.base-url=https://api.deepseek.com
spring.ai.openai.chat.options.model=deepseek-chat
spring.ai.openai.chat.options.temperature=0.7
```

**配置说明**：
- `api-key`: 你刚才复制的DeepSeek API Key
- `base-url`: DeepSeek的API地址
- `model`: 使用的模型（deepseek-chat是通用对话模型）
- `temperature`: 控制回答的创造性（0-1，越高越有创意）

#### 创建Controller

在项目中创建 `ChatController.java`：

```java
package com.example.aidemo.controller;

import org.springframework.ai.chat.client.ChatClient;
import org.springframework.web.bind.annotation.GetMapping;
import org.springframework.web.bind.annotation.RequestParam;
import org.springframework.web.bind.annotation.RestController;

@RestController
public class ChatController {

    private final ChatClient chatClient;

    // 通过构造器注入ChatClient
    public ChatController(ChatClient.Builder chatClientBuilder) {
        this.chatClient = chatClientBuilder.build();
    }

    @GetMapping("/ai/chat")
    public String chat(@RequestParam(value = "msg", defaultValue = "你好") String message) {
        // 调用AI模型
        return chatClient.prompt()
                .user(message)  // 用户输入的问题
                .call()         // 发起请求
                .content();     // 获取回答内容
    }
}
```

**代码解释**：
- `@RestController`: 标记这是一个REST接口
- `ChatClient`: Spring AI提供的聊天客户端
- `prompt().user().call().content()`: 链式调用，发送问题并获取回答

#### 创建启动类

确保有启动类 `AiDemoApplication.java`：

```java
package com.example.aidemo;

import org.springframework.boot.SpringApplication;
import org.springframework.boot.autoconfigure.SpringBootApplication;

@SpringBootApplication
public class AiDemoApplication {
    public static void main(String[] args) {
        SpringApplication.run(AiDemoApplication.class, args);
    }
}
```

## 🐛 常见问题排查

### 问题1：启动报错 "ClassNotFoundException"

**原因**：依赖未下载完成

**解决**：
```bash
mvn clean install
```

### 问题2：访问接口返回401错误

**原因**：API Key配置错误或无效

**解决**：
- 检查 `application.properties` 中的 `api-key` 是否正确
- 确认API Key没有多余空格
- 验证API Key是否有效（登录DeepSeek控制台查看）

### 问题3：访问接口超时

**原因**：网络连接问题或API限流

**解决**：
- 检查网络连接
- 确认可以访问 https://api.deepseek.com
- 稍后再试（可能是临时限流）

### 问题4：中文乱码

**原因**：响应编码问题

**解决**：在Controller方法上添加：
```java
@GetMapping(value = "/ai/chat", produces = "text/plain;charset=UTF-8")
```
