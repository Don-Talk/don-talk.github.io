## 🎯 学习路径

### 📘 初级阶段（2周）

**目标**：掌握基本的构建和版本控制

1. **Maven 基础**
   - POM 文件结构
   - 依赖管理
   - 生命周期

2. **Git 基础**
   - 基本命令
   - 分支管理
   - SSH 密钥配置

### 📗 中级阶段（1个月）

**目标**：实现自动化构建和持续集成

3. **Jenkins CI/CD**
   - Jenkins 安装配置
   - Pipeline 编写
   - 自动化部署

4. **Nginx 基础**
   - 反向代理
   - 负载均衡
   - 静态资源服务

### 📙 高级阶段（1-2个月）

**目标**：掌握企业级工程化实践

5. **配置中心**
   - Apollo 架构
   - 动态配置
   - 灰度发布

6. **DevOps 实践**
   - 容器化部署
   - 监控告警
   - 日志收集

## 💡 学习建议

### 1. 工具链整合

- Maven + Git + Jenkins 形成完整工具链
- 理解每个工具的定位和作用
- 学会配置和优化

### 2. 自动化思维

- 尽可能自动化重复工作
- 编写可复用的脚本
- 建立标准化流程

### 3. 最佳实践

- 遵循行业标准和规范
- 参考开源项目的实践
- 持续优化和改进

### 4. 安全意识

- 敏感信息加密存储
- 权限控制和审计
- 安全扫描和检测

## 🛠️ 常用工具

### 构建工具
- **Maven**：Java 项目管理
- **Gradle**：现代化构建工具
- **Nexus**：Maven 私服

### CI/CD
- **Jenkins**：持续集成服务器
- **GitLab CI**：GitLab 内置 CI
- **GitHub Actions**：GitHub 自动化

### 配置管理
- **Apollo**：携程开源配置中心
- **Nacos**：阿里配置中心
- **Consul**：HashiCorp 配置中心

### Web 服务器
- **Nginx**：高性能 Web 服务器
- **Apache**：传统 Web 服务器
- **Caddy**：现代化 Web 服务器
