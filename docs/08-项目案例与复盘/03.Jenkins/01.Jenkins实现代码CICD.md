# Jenkins实现代码CICD



#### **步骤 1：安装 Docker**

#### **步骤 2：启动 Jenkins 容器**

```shell
# 创建 Jenkins 数据目录（持久化存储配置和构建记录）
sudo mkdir -p /var/jenkins_home
sudo chmod 777 /var/jenkins_home  # 确保容器有写入权限

# 运行 Jenkins 容器（关键参数说明）：
sudo docker run -d \
  --name jenkins_watch \
  -p 6500:8080 \          # 访问端口
  -p 6501:50000 \        # Agent 通信端口
  -v /opt/docker/jenkins/jenkins_home:/var/jenkins_home \  # 数据持久化
  -v /var/run/docker.sock:/var/run/docker.sock \  # 允许容器内调用宿主机 Docker
  --restart=always \      # 自动重启
  jenkins/jenkins:lts
  
docker run -d --name jenkins_watch -p 6500:8080 -p 6501:50000  -v /opt/docker/jenkins/jenkins_home:/var/jenkins_home --restart=always  jenkins/jenkins:lts

# 查看初始管理员密码（首次启动后生成）
sudo docker logs jenkins  # 在日志中查找 `Admin password: xxx`
