## 🎯 学习路径

### 📘 初级阶段（1个月）

**目标**：掌握基础的服务器运维技能

1. **Linux 基础**
   - 常用命令
   - 文件权限
   - 用户管理

2. **FTP 服务器搭建**
   - vsftpd 安装配置
   - 被动模式设置
   - 防火墙配置

### 📗 中级阶段（1-2个月）

**目标**：实现自动化部署和运维

3. **Jenkins CI/CD**
   - Jenkins 安装
   - Pipeline 编写
   - 自动化部署

4. **Docker 容器化**
   - Docker 基础
   - 镜像制作
   - 容器编排

### 📙 高级阶段（2-3个月）

**目标**：掌握云原生和复杂系统设计

5. **Kubernetes**
   - K8s 架构
   - 资源对象
   - 服务部署

6. **业务系统开发**
   - 回单处理系统
   - PDF 处理
   - 银行接口对接

## 💡 学习建议

### 1. 从简单开始

- 先掌握 Linux 基础命令
- 再学习服务搭建
- 逐步增加复杂度

### 2. 动手实践

- 每个案例都要亲自实践
- 记录遇到的问题和解决方案
- 建立自己的知识库

### 3. 理解原理

- 不仅会用，还要理解为什么
- 阅读官方文档
- 学习最佳实践

### 4. 总结经验

- 每次项目后做技术复盘
- 总结成功经验和失败教训
- 形成可复用的方法论
