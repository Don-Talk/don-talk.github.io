## 一、Specification动态查询

### 1.1 什么是Specification

**Specification** 是JPA提供的动态查询API,用于构建复杂的查询条件。

**适用场景**:
- 多条件组合查询
- 条件可选的搜索功能
- 需要动态拼接SQL的场景

### 1.2 基本用法

#### Repository接口

```java
public interface UserRepository extends JpaRepository<User, Long>, 
                                       JpaSpecificationExecutor<User> {
    // 继承JpaSpecificationExecutor获得Specification支持
}
```

#### Service层构建查询

```java
@Service
public class UserService {
    
    @Autowired
    private UserRepository userRepository;
    
    public List<User> searchUsers(String username, Integer minAge, Integer maxAge) {
        
        Specification<User> spec = (root, query, cb) -> {
            List<Predicate> predicates = new ArrayList<>();
            
            // 用户名模糊查询
            if (StringUtils.isNotBlank(username)) {
                predicates.add(cb.like(root.get("username"), "%" + username + "%"));
            }
            
            // 年龄范围查询
            if (minAge != null) {
                predicates.add(cb.greaterThanOrEqualTo(root.get("age"), minAge));
            }
            if (maxAge != null) {
                predicates.add(cb.lessThanOrEqualTo(root.get("age"), maxAge));
            }
            
            return cb.and(predicates.toArray(new Predicate[0]));
        };
        
        return userRepository.findAll(spec);
    }
}
```

### 1.3 复杂查询示例

#### 多表关联查询

```java
public List<Order> searchOrders(Long userId, String orderNo, 
                                BigDecimal minAmount, Date startDate) {
    
    Specification<Order> spec = (root, query, cb) -> {
        List<Predicate> predicates = new ArrayList<>();
        
        // 用户ID
        if (userId != null) {
            predicates.add(cb.equal(root.get("user").get("id"), userId));
        }
        
        // 订单号模糊查询
        if (StringUtils.isNotBlank(orderNo)) {
            predicates.add(cb.like(root.get("orderNo"), "%" + orderNo + "%"));
        }
        
        // 金额范围
        if (minAmount != null) {
            predicates.add(cb.greaterThanOrEqualTo(root.get("amount"), minAmount));
        }
        
        // 日期范围
        if (startDate != null) {
            predicates.add(cb.greaterThanOrEqualTo(root.get("createTime"), startDate));
        }
        
        // 按创建时间降序
        query.orderBy(cb.desc(root.get("createTime")));
        
        return cb.and(predicates.toArray(new Predicate[0]));
    };
    
    return orderRepository.findAll(spec);
}
```

#### 分页+排序+Specification

```java
public Page<User> searchUsersWithPage(String keyword, int page, int size) {
    
    Specification<User> spec = (root, query, cb) -> {
        return cb.or(
            cb.like(root.get("username"), "%" + keyword + "%"),
            cb.like(root.get("email"), "%" + keyword + "%")
        );
    };
    
    Pageable pageable = PageRequest.of(page, size, Sort.by("createTime").descending());
    
    return userRepository.findAll(spec, pageable);
}
```

### 1.4 封装通用Specification工具类

```java
public class SpecificationUtil {
    
    /**
     * 等于
     */
    public static <T> Specification<T> equal(String field, Object value) {
        return (root, query, cb) -> value == null ? null : cb.equal(root.get(field), value);
    }
    
    /**
     * 模糊查询
     */
    public static <T> Specification<T> like(String field, String value) {
        return (root, query, cb) -> StringUtils.isBlank(value) ? null : 
            cb.like(root.get(field), "%" + value + "%");
    }
    
    /**
     * 大于等于
     */
    public static <T> Specification<T> ge(String field, Comparable value) {
        return (root, query, cb) -> value == null ? null : 
            cb.greaterThanOrEqualTo(root.get(field), value);
    }
    
    /**
     * 小于等于
     */
    public static <T> Specification<T> le(String field, Comparable value) {
        return (root, query, cb) -> value == null ? null : 
            cb.lessThanOrEqualTo(root.get(field), value);
    }
    
    /**
     * 组合AND
     */
    @SafeVarargs
    public static <T> Specification<T> and(Specification<T>... specs) {
        return (root, query, cb) -> {
            List<Predicate> predicates = new ArrayList<>();
            for (Specification<T> spec : specs) {
                if (spec != null) {
                    Predicate predicate = spec.toPredicate(root, query, cb);
                    if (predicate != null) {
                        predicates.add(predicate);
                    }
                }
            }
            return predicates.isEmpty() ? null : cb.and(predicates.toArray(new Predicate[0]));
        };
    }
}
```

**使用**:

```java
Specification<User> spec = SpecificationUtil.and(
    SpecificationUtil.like("username", keyword),
    SpecificationUtil.ge("age", 18),
    SpecificationUtil.le("age", 30)
);

List<User> users = userRepository.findAll(spec);
```

## 二、Query By Example (QBE)

### 2.1 什么是QBE

**QBE (Query By Example)** 通过示例对象进行查询,适合简单条件查询。

### 2.2 基本用法

```java
@Service
public class UserService {
    
    @Autowired
    private UserRepository userRepository;
    
    public List<User> searchByExample(User exampleUser) {
        
        // 创建Example
        Example<User> example = Example.of(exampleUser);
        
        return userRepository.findAll(example);
    }
}
```

**调用**:

```java
User probe = new User();
probe.setUsername("张");  // 模糊匹配
probe.setAge(20);

List<User> users = userService.searchByExample(probe);
// SELECT * FROM user WHERE username LIKE '%张%' AND age = 20
```

### 2.3 配置ExampleMatcher

```java
// 创建匹配器
ExampleMatcher matcher = ExampleMatcher.matching()
    .withIgnoreCase()              // 忽略大小写
    .withStringMatcher(ExampleMatcher.StringMatcher.CONTAINING)  // 包含匹配
    .withIgnoreNullValues();       // 忽略null值

// 创建Example
User probe = new User();
probe.setUsername("zhang");

Example<User> example = Example.of(probe, matcher);

List<User> users = userRepository.findAll(example);
```

**匹配模式**:

| 模式 | 说明 | SQL |
|------|------|-----|
| DEFAULT | 精确匹配 | WHERE username = 'zhang' |
| EXACT | 精确匹配(区分大小写) | WHERE username = 'zhang' |
| STARTING | 开头匹配 | WHERE username LIKE 'zhang%' |
| ENDING | 结尾匹配 | WHERE username LIKE '%zhang' |
| CONTAINING | 包含匹配 | WHERE username LIKE '%zhang%' |

## 三、投影(Projection)

### 3.1 为什么需要投影

**问题**: 只需要部分字段,但查询返回整个实体,浪费性能。

```java
// 只需要用户名和邮箱,但查询了所有字段
List<User> users = userRepository.findAll();
```

**解决**: 使用投影只查询需要的字段。

### 3.2 接口-based投影

#### 定义投影接口

```java
public interface UserSummary {
    String getUsername();
    String getEmail();
}
```

#### Repository方法

```java
public interface UserRepository extends JpaRepository<User, Long> {
    
    List<UserSummary> findAllBy();
    
    UserSummary findByUsername(String username);
}
```

**效果**: 只查询username和email字段。

### 3.3 DTO-based投影

#### 创建DTO

```java
@Data
@AllArgsConstructor
public class UserDTO {
    private String username;
    private String email;
    private Integer age;
}
```

#### Repository方法

```java
public interface UserRepository extends JpaRepository<User, Long> {
    
    @Query("SELECT NEW com.example.dto.UserDTO(u.username, u.email, u.age) " +
           "FROM User u WHERE u.age > :age")
    List<UserDTO> findUserDTOs(@Param("age") Integer age);
}
```

### 3.4 动态投影

```java
public interface UserRepository extends JpaRepository<User, Long> {
    
    <T> List<T> findAllBy(Class<T> type);
}
```

**使用**:

```java
// 返回完整实体
List<User> users = userRepository.findAllBy(User.class);

// 返回投影
List<UserSummary> summaries = userRepository.findAllBy(UserSummary.class);
```

## 四、审计功能(Auditing)

### 4.1 自动记录时间

#### 启用审计

```java
@SpringBootApplication
@EnableJpaAuditing  // 启用JPA审计
public class Application {
    public static void main(String[] args) {
        SpringApplication.run(Application.class, args);
    }
}
```

#### 实体类添加审计字段

```java
@Entity
@Table(name = "user")
@EntityListeners(AuditingEntityListener.class)  // 启用审计监听器
public class User {
    
    @Id
    @GeneratedValue(strategy = GenerationType.IDENTITY)
    private Long id;
    
    private String username;
    
    // 创建时间(自动填充)
    @CreatedDate
    @Column(name = "create_time", updatable = false)
    private LocalDateTime createTime;
    
    // 更新时间(自动更新)
    @LastModifiedDate
    @Column(name = "update_time")
    private LocalDateTime updateTime;
    
    // 创建人
    @CreatedBy
    private String createdBy;
    
    // 更新人
    @LastModifiedBy
    private String updatedBy;
}
```

#### 配置审计人员

```java
@Configuration
public class AuditorConfig {
    
    @Bean
    public AuditorAware<String> auditorAware() {
        return () -> {
            // 从SecurityContext获取当前用户
            Authentication authentication = SecurityContextHolder.getContext().getAuthentication();
            if (authentication != null && authentication.isAuthenticated()) {
                return authentication.getName();
            }
            return "system";
        };
    }
}
```

**效果**:
- 保存时自动设置createTime和createdBy
- 更新时自动设置updateTime和updatedBy

## 五、事务管理

### 5.1 @Transactional注解

```java
@Service
@Transactional  // 类级别,所有方法都有事务
public class OrderService {
    
    @Autowired
    private OrderRepository orderRepository;
    
    @Autowired
    private StockRepository stockRepository;
    
    /**
     * 创建订单(事务方法)
     */
    @Transactional(rollbackFor = Exception.class)
    public Long createOrder(OrderDTO orderDTO) {
        
        // 1. 创建订单
        Order order = new Order();
        order.setOrderNo(generateOrderNo());
        order.setAmount(orderDTO.getAmount());
        order = orderRepository.save(order);
        
        // 2. 扣减库存
        Stock stock = stockRepository.findById(orderDTO.getProductId())
            .orElseThrow(() -> new RuntimeException("商品不存在"));
        
        if (stock.getQuantity() < orderDTO.getQuantity()) {
            throw new RuntimeException("库存不足");
        }
        
        stock.setQuantity(stock.getQuantity() - orderDTO.getQuantity());
        stockRepository.save(stock);
        
        return order.getId();
    }
}
```

### 5.2 事务传播行为

```java
@Service
public class BatchService {
    
    @Autowired
    private OrderService orderService;
    
    @Transactional
    public void batchCreateOrders(List<OrderDTO> orders) {
        for (OrderDTO orderDTO : orders) {
            try {
                // REQUIRES_NEW: 每个订单独立事务
                orderService.createOrderWithNewTransaction(orderDTO);
            } catch (Exception e) {
                log.error("订单创建失败", e);
                // 单个失败不影响其他订单
            }
        }
    }
}

@Service
public class OrderService {
    
    @Transactional(propagation = Propagation.REQUIRES_NEW)
    public Long createOrderWithNewTransaction(OrderDTO orderDTO) {
        // 独立事务,不受外层影响
        return createOrder(orderDTO);
    }
}
```

## 六、乐观锁

### 6.1 版本控制

#### 数据库添加version字段

```sql
ALTER TABLE product ADD COLUMN version INT DEFAULT 0;
```

#### 实体类

```java
@Entity
@Table(name = "product")
public class Product {
    
    @Id
    @GeneratedValue(strategy = GenerationType.IDENTITY)
    private Long id;
    
    private String productName;
    
    private Integer stock;
    
    // 版本号
    @Version
    private Integer version;
}
```

#### 使用

```java
@Service
public class ProductService {
    
    @Autowired
    private ProductRepository productRepository;
    
    @Transactional
    public void decreaseStock(Long productId, Integer quantity) {
        
        Product product = productRepository.findById(productId)
            .orElseThrow(() -> new RuntimeException("商品不存在"));
        
        if (product.getStock() < quantity) {
            throw new RuntimeException("库存不足");
        }
        
        product.setStock(product.getStock() - quantity);
        
        // 更新时会自动检查version
        // UPDATE product SET stock=?, version=version+1 
        // WHERE id=? AND version=?
        productRepository.save(product);
    }
}
```

**并发控制**:
- 两个线程同时读取version=1
- 线程A先更新,version变为2
- 线程B更新时发现version不匹配,抛出OptimisticLockingFailureException
