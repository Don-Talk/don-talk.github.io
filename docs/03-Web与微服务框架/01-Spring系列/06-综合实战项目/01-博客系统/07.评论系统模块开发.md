## 一、评论实体类

```java
package com.dontalk.blog.module.comment.entity;

import com.baomidou.mybatisplus.annotation.TableName;
import com.dontalk.blog.common.entity.BaseEntity;
import io.swagger.annotations.ApiModel;
import io.swagger.annotations.ApiModelProperty;
import lombok.Data;
import lombok.EqualsAndHashCode;

@Data
@EqualsAndHashCode(callSuper = true)
@TableName("comment")
@ApiModel(value = "Comment对象", description = "评论实体")
public class Comment extends BaseEntity {
    
    private static final long serialVersionUID = 1L;
    
    @ApiModelProperty("文章ID")
    private Long articleId;
    
    @ApiModelProperty("评论人ID")
    private Long userId;
    
    @ApiModelProperty("评论内容")
    private String content;
    
    @ApiModelProperty("父评论ID(回复时填写)")
    private Long parentId;
    
    @ApiModelProperty("被回复的评论ID")
    private Long replyToId;
    
    @ApiModelProperty("点赞数")
    private Integer likeCount;
    
    @ApiModelProperty("状态: 0-待审核, 1-已通过, 2-已拒绝")

}
```

**字段说明**:
- `parentId`: 用于构建评论树(一级评论为null)
- `replyToId`: 记录具体回复哪条评论
- `status`: 支持评论审核机制

### 2.1 CommentMapper.java

```java
package com.dontalk.blog.module.comment.mapper;

import com.baomidou.mybatisplus.core.mapper.BaseMapper;
import com.dontalk.blog.module.comment.entity.Comment;
import org.apache.ibatis.annotations.Mapper;
import org.apache.ibatis.annotations.Param;
import org.apache.ibatis.annotations.Select;

import java.util.List;

@Mapper
public interface CommentMapper extends BaseMapper<Comment> {
    
    /**
     * 查询文章的评论列表(只查一级评论)
     */
    @Select("SELECT * FROM comment WHERE article_id = #{articleId} AND parent_id IS NULL " +
            "AND status = 1 ORDER BY create_time DESC")
    List<Comment> findTopComments(Long articleId);
    
    /**
     * 查询某条评论的所有回复
     */
    @Select("SELECT * FROM comment WHERE parent_id = #{parentId} AND status = 1 " +
            "ORDER BY create_time ASC")
    List<Comment> findReplies(Long parentId);
    
    /**
     * 统计文章的评论数
     */
    @Select("SELECT COUNT(*) FROM comment WHERE article_id = #{articleId} AND status = 1")
    int countByArticleId(Long articleId);
    
    /**
     * 增加点赞数
     */
    void incrementLikeCount(@Param("id") Long id);
}
```

```java
package com.dontalk.blog.module.comment.service.impl;

import com.baomidou.mybatisplus.core.conditions.query.LambdaQueryWrapper;
import com.dontalk.blog.common.exception.BusinessException;
import com.dontalk.blog.module.article.entity.Article;
import com.dontalk.blog.module.article.mapper.ArticleMapper;
import com.dontalk.blog.module.comment.dto.CommentCreateRequest;
import com.dontalk.blog.module.comment.entity.Comment;
import com.dontalk.blog.module.comment.mapper.CommentMapper;
import com.dontalk.blog.module.comment.service.CommentService;
import com.dontalk.blog.module.comment.vo.CommentResponse;
import com.dontalk.blog.module.user.entity.User;
import com.dontalk.blog.module.user.mapper.UserMapper;
import lombok.RequiredArgsConstructor;
import lombok.extern.slf4j.Slf4j;
import org.springframework.beans.BeanUtils;
import org.springframework.stereotype.Service;
import org.springframework.transaction.annotation.Transactional;

import java.util.ArrayList;
import java.util.List;
import java.util.Map;
import java.util.stream.Collectors;

@Slf4j
@Service
@RequiredArgsConstructor
public class CommentServiceImpl implements CommentService {
    
    private final CommentMapper commentMapper;
    private final ArticleMapper articleMapper;
    private final UserMapper userMapper;
    
    @Override
    @Transactional(rollbackFor = Exception.class)
    public Long createComment(CommentCreateRequest request, Long userId) {
        // 1. 验证文章是否存在
        Article article = articleMapper.selectById(request.getArticleId());
        if (article == null) {
            throw new BusinessException("文章不存在");
        }
        
        // 2. 如果是回复评论，验证父评论是否存在
        if (request.getParentId() != null) {
            Comment parentComment = commentMapper.selectById(request.getParentId());
            if (parentComment == null) {
                throw new BusinessException("父评论不存在");
            }
        }
        
        // 3. 构建评论实体
        Comment comment = new Comment();
        BeanUtils.copyProperties(request, comment);
        comment.setUserId(userId);
        comment.setLikeCount(0);
        comment.setStatus(1); // 默认直接通过
        
        // 4. 保存评论
        commentMapper.insert(comment);
        
        // 5. 更新文章评论数
        updateArticleCommentCount(request.getArticleId());
        
        log.info("发表评论成功: id={}, articleId={}", comment.getId(), request.getArticleId());
        return comment.getId();
    }
    
    @Override
    @Transactional(rollbackFor = Exception.class)
    public void deleteComment(Long id, Long userId) {
        // 1. 查询评论是否存在
        Comment comment = commentMapper.selectById(id);
        if (comment == null) {
            throw new BusinessException("评论不存在");
        }
        
        // 2. 权限校验: 只能删除自己的评论
        if (!comment.getUserId().equals(userId)) {
            throw new BusinessException("无权删除他人评论");
        }
        
        // 3. 逻辑删除评论
        comment.setStatus(2);
        commentMapper.updateById(comment);
        
        // 4. 更新文章评论数
        updateArticleCommentCount(comment.getArticleId());
        
        log.info("删除评论成功: id={}", id);
    }
    
    @Override
    public List<CommentResponse> getCommentsByArticleId(Long articleId) {
        // 1. 查询所有顶级评论(parentId为null)
        List<Comment> topComments = commentMapper.findTopComments(articleId);
        
        // 2. 转换为VO并填充回复
        return topComments.stream()
                .map(this::convertToResponseWithReplies)
                .collect(Collectors.toList());
    }
    
    @Override
    @Transactional(rollbackFor = Exception.class)
    public void likeComment(Long commentId, Long userId) {
        // 1. 查询评论是否存在
        Comment comment = commentMapper.selectById(commentId);
        if (comment == null) {
            throw new BusinessException("评论不存在");
        }
        
        // 2. TODO: 检查是否已经点赞过(需要comment_like表)
        // LambdaQueryWrapper<CommentLike> wrapper = new LambdaQueryWrapper<>();
        // wrapper.eq(CommentLike::getCommentId, commentId)
        //        .eq(CommentLike::getUserId, userId);
        // if (commentLikeMapper.selectCount(wrapper) > 0) {
        //     throw new BusinessException("已经点赞过");
        // }
        
        // 3. 增加点赞数
        commentMapper.incrementLikeCount(commentId);
        
        // 4. TODO: 插入点赞记录
        // commentLikeMapper.insert(new CommentLike(commentId, userId));
        
        log.info("点赞评论成功: commentId={}, userId={}", commentId, userId);
    }
    
    @Override
    @Transactional(rollbackFor = Exception.class)
    public void unlikeComment(Long commentId, Long userId) {
        // 1. 查询评论是否存在
        Comment comment = commentMapper.selectById(commentId);
        if (comment == null) {
            throw new BusinessException("评论不存在");
        }
        
        // 2. TODO: 删除点赞记录
        // LambdaQueryWrapper<CommentLike> wrapper = new LambdaQueryWrapper<>();
        // wrapper.eq(CommentLike::getCommentId, commentId)
        //        .eq(CommentLike::getUserId, userId);
        // commentLikeMapper.delete(wrapper);
        
        // 3. 减少点赞数
        // TODO: 需要decrementLikeCount方法
        
        log.info("取消点赞成功: commentId={}, userId={}", commentId, userId);
    }
    
    // ==================== 私有方法 ====================
    
    /**
     * 转换为响应VO(包含回复列表)
     */
    private CommentResponse convertToResponseWithReplies(Comment comment) {
        CommentResponse response = convertToResponse(comment);
        
        // 查询该评论的所有回复
        List<Comment> replies = commentMapper.findReplies(comment.getId());
        if (!replies.isEmpty()) {
            response.setReplies(replies.stream()
                    .map(this::convertToResponse)
                    .collect(Collectors.toList()));
        }
        
        return response;
    }
    
    /**
     * 转换为响应VO
     */
    private CommentResponse convertToResponse(Comment comment) {
        CommentResponse response = new CommentResponse();
        BeanUtils.copyProperties(comment, response);
        
        // 填充评论人信息
        User user = userMapper.selectById(comment.getUserId());
        if (user != null) {
            response.setUserName(user.getNickname());
            response.setUserAvatar(user.getAvatar());
        }
        
        // 填充被回复人信息
        if (comment.getReplyToId() != null) {
            Comment replyToComment = commentMapper.selectById(comment.getReplyToId());
            if (replyToComment != null) {
                User replyToUser = userMapper.selectById(replyToComment.getUserId());
                if (replyToUser != null) {
                    response.setReplyToUserName(replyToUser.getNickname());
                }
            }
        }
        
        // TODO: 设置liked字段(需要查询comment_like表)
        response.setLiked(false);
        
        return response;
    }
    
    /**
     * 更新文章评论数
     */
    private void updateArticleCommentCount(Long articleId) {
        int count = commentMapper.countByArticleId(articleId);
        
        Article article = articleMapper.selectById(articleId);
        if (article != null) {
            article.setCommentCount(count);
            articleMapper.updateById(article);
        }
    }
}
```

**关键逻辑说明**:
1. **树形结构**: 通过`parentId`和`findReplies()`构建评论树
2. **权限控制**: 只能删除自己的评论
3. **数据一致性**: 评论增删时同步更新文章评论数
4. **TODO标记**: 点赞功能需要comment_like关联表

## 五、Controller层

### 5.1 CommentController.java

```java
package com.dontalk.blog.module.comment.controller;

import com.dontalk.blog.common.result.Result;
import com.dontalk.blog.module.comment.dto.CommentCreateRequest;
import com.dontalk.blog.module.comment.service.CommentService;
import com.dontalk.blog.module.comment.vo.CommentResponse;
import com.dontalk.blog.security.util.SecurityUtils;
import io.swagger.annotations.Api;
import io.swagger.annotations.ApiOperation;
import lombok.RequiredArgsConstructor;
import org.springframework.validation.annotation.Validated;
import org.springframework.web.bind.annotation.*;

import java.util.List;

@RestController
@RequestMapping("/api/comments")
@RequiredArgsConstructor
@Api(tags = "评论管理")
public class CommentController {
    
    private final CommentService commentService;
    
    @PostMapping
    @ApiOperation("发表评论")
    public Result<Long> createComment(@Validated @RequestBody CommentCreateRequest request) {
        Long userId = SecurityUtils.getCurrentUserId();
        Long commentId = commentService.createComment(request, userId);
        return Result.success(commentId);
    }
    
    @DeleteMapping("/{id}")
    @ApiOperation("删除评论")
    public Result<Void> deleteComment(@PathVariable Long id) {
        Long userId = SecurityUtils.getCurrentUserId();
        commentService.deleteComment(id, userId);
        return Result.success();
    }
    
    @GetMapping("/article/{articleId}")
    @ApiOperation("查询文章评论列表")
    public Result<List<CommentResponse>> getComments(@PathVariable Long articleId) {
        List<CommentResponse> comments = commentService.getCommentsByArticleId(articleId);
        return Result.success(comments);
    }
    
    @PostMapping("/{id}/like")
    @ApiOperation("点赞评论")
    public Result<Void> likeComment(@PathVariable Long id) {
        Long userId = SecurityUtils.getCurrentUserId();
        commentService.likeComment(id, userId);
        return Result.success();
    }
    
    @DeleteMapping("/{id}/like")
    @ApiOperation("取消点赞")
    public Result<Void> unlikeComment(@PathVariable Long id) {
        Long userId = SecurityUtils.getCurrentUserId();
        commentService.unlikeComment(id, userId);
        return Result.success();
    }
}
```

## 六、数据库表设计

### 6.1 创建表的SQL

```sql
-- 评论表
CREATE TABLE `comment` (
  `id` bigint NOT NULL AUTO_INCREMENT COMMENT '主键ID',
  `article_id` bigint NOT NULL COMMENT '文章ID',
  `user_id` bigint NOT NULL COMMENT '评论人ID',
  `content` text NOT NULL COMMENT '评论内容',
  `parent_id` bigint DEFAULT NULL COMMENT '父评论ID',
  `reply_to_id` bigint DEFAULT NULL COMMENT '被回复的评论ID',
  `like_count` int DEFAULT '0' COMMENT '点赞数',
  `status` tinyint DEFAULT '1' COMMENT '状态: 0-待审核, 1-已通过, 2-已拒绝',
  `create_time` datetime DEFAULT CURRENT_TIMESTAMP COMMENT '创建时间',
  `update_time` datetime DEFAULT CURRENT_TIMESTAMP ON UPDATE CURRENT_TIMESTAMP COMMENT '更新时间',
  PRIMARY KEY (`id`),
  KEY `idx_article_id` (`article_id`),
  KEY `idx_user_id` (`user_id`),
  KEY `idx_parent_id` (`parent_id`)
) ENGINE=InnoDB DEFAULT CHARSET=utf8mb4 COMMENT='评论表';

-- 评论点赞表
CREATE TABLE `comment_like` (
  `id` bigint NOT NULL AUTO_INCREMENT COMMENT '主键ID',
  `comment_id` bigint NOT NULL COMMENT '评论ID',
  `user_id` bigint NOT NULL COMMENT '用户ID',
  `create_time` datetime DEFAULT CURRENT_TIMESTAMP COMMENT '创建时间',
  PRIMARY KEY (`id`),
  UNIQUE KEY `uk_comment_user` (`comment_id`, `user_id`),
  KEY `idx_user_id` (`user_id`)
) ENGINE=InnoDB DEFAULT CHARSET=utf8mb4 COMMENT='评论点赞表';
```

## 九、常见问题

### Q1: 如何实现评论的分页加载?

**A**: 对于大量评论的场景:
```java
// 分页查询顶级评论
Page<Comment> page = new Page<>(pageNum, pageSize);
LambdaQueryWrapper<Comment> wrapper = new LambdaQueryWrapper<>();
wrapper.eq(Comment::getArticleId, articleId)
       .isNull(Comment::getParentId)
       .eq(Comment::getStatus, 1)
       .orderByDesc(Comment::getCreateTime);
Page<Comment> commentPage = commentMapper.selectPage(page, wrapper);
```

### Q2: 如何防止恶意刷评论?

**A**: 
1. **频率限制**: 同一用户每分钟最多发5条评论
2. **内容审核**: 敏感词过滤
3. **验证码**: 提交评论前需要滑块验证
4. **IP限流**: 同一IP每小时最多发50条评论

### Q3: 如何实现@提及功能?

**A**: 
1. 解析评论内容中的`@用户名`
2. 查询用户ID
3. 在数据库中记录被提及的用户
4. 发送通知给被提及的用户
