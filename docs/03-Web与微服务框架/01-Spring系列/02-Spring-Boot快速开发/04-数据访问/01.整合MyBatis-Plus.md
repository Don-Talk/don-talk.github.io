## 一、整合方式概述

### 1.1 两种整合方案

#### 方案一: mybatis-spring-boot-starter (官方)

```xml
<dependency>
    <groupId>org.mybatis.spring.boot</groupId>
    <artifactId>mybatis-spring-boot-starter</artifactId>
    <version>2.3.0</version>
</dependency>
```

**特点**:
- MyBatis官方提供
- 轻量级,只包含核心功能
- 需要手动编写SQL

#### 方案二: mybatis-plus-boot-starter (推荐)

```xml
<dependency>
    <groupId>com.baomidou</groupId>
    <artifactId>mybatis-plus-boot-starter</artifactId>
    <version>3.5.3</version>
</dependency>
```

**特点**:
- 在MyBatis基础上增强
- 提供通用CRUD,无需编写SQL
- 内置分页、代码生成器等
- **推荐使用**

## 二、MyBatis-Plus快速入门

### 2.1 环境搭建

#### 第一步: 导入依赖

```xml
<dependencies>
    <!--Spring Boot Web-->
    <dependency>

        <artifactId>spring-boot-starter-web</artifactId>
    </dependency>
    
    <!--MyBatis-Plus-->
    <dependency>
        <groupId>com.baomidou</groupId>
        <artifactId>mybatis-plus-boot-starter</artifactId>
        <version>3.5.3</version>

    <dependency>
        <groupId>mysql</groupId>
        <artifactId>mysql-connector-java</artifactId>

    <!--Lombok-->
    <dependency>
        <groupId>org.projectlombok</groupId>
        <artifactId>lombok</artifactId>
        <optional>true</optional>
    </dependency>

```

#### 第二步: 配置数据源

`application.yml`:

```yaml
spring:
  datasource:
    driver-class-name: com.mysql.cj.jdbc.Driver
    url: jdbc:mysql://localhost:3306/mydb?useUnicode=true&characterEncoding=utf8&serverTimezone=Asia/Shanghai
    username: root
    password: root

# MyBatis-Plus配置
mybatis-plus:
  # Mapper XML文件位置
  mapper-locations: classpath:mapper/*.xml

    # 驼峰命名转换
    map-underscore-to-camel-case: true
```

#### 第三步: 创建数据库表

```sql

  `id` BIGINT NOT NULL AUTO_INCREMENT COMMENT '用户ID',
```

## 三、条件构造器Wrapper

### 3.1 QueryWrapper (查询条件)

```java
// 示例1: 简单条件
List<User> users = userService.lambdaQuery()
    .eq(User::getAge, 20)              // age = 20
    .like(User::getUsername, "张")     // username LIKE '%张%'
    .orderByDesc(User::getCreateTime)  // ORDER BY create_time DESC
    .list();

// 示例2: 复杂条件
List<User> users = userService.lambdaQuery()
    .ge(User::getAge, 18)                      // age >= 18
    .le(User::getAge, 30)                      // age <= 30
    .in(User::getId, 1, 2, 3)                  // id IN (1,2,3)
    .isNotNull(User::getEmail)                 // email IS NOT NULL
    .between(User::getAge, 20, 30)             // age BETWEEN 20 AND 30
    .list();

// 示例3: OR条件
List<User> users = userService.lambdaQuery()
    .eq(User::getAge, 20)
    .or()
    .eq(User::getAge, 25)
    .list();

// 示例4: 动态条件
String username = "张三";
Integer age = 20;

LambdaQueryWrapper<User> wrapper = new LambdaQueryWrapper<>();
wrapper.eq(StringUtils.isNotBlank(username), User::getUsername, username)
       .eq(age != null, User::getAge, age);

List<User> users = userService.list(wrapper);
```

### 3.2 UpdateWrapper (更新条件)

```java
// 批量更新
boolean success = userService.lambdaUpdate()
    .set(User::getEmail, "new@example.com")
    .eq(User::getAge, 20)
    .update();

// 条件删除
boolean success = userService.lambdaUpdate()
    .set(User::getDeleted, 1)
    .eq(User::getAge, 100)
    .update();
```

### 3.3 常用条件方法

| 方法 | 说明 | SQL示例 |
|------|------|---------|
| eq | 等于 | WHERE age = 20 |
| ne | 不等于 | WHERE age <> 20 |
| gt | 大于 | WHERE age > 20 |
| ge | 大于等于 | WHERE age >= 20 |
| lt | 小于 | WHERE age < 20 |
| le | 小于等于 | WHERE age <= 20 |
| like | 模糊查询 | WHERE name LIKE '%张%' |
| notLike | 不匹配 | WHERE name NOT LIKE '%张%' |
| in | IN查询 | WHERE id IN (1,2,3) |
| notIn | NOT IN | WHERE id NOT IN (1,2,3) |
| between | BETWEEN | WHERE age BETWEEN 20 AND 30 |
| isNull | IS NULL | WHERE email IS NULL |
| isNotNull | IS NOT NULL | WHERE email IS NOT NULL |
| orderByAsc | 升序 | ORDER BY age ASC |
| orderByDesc | 降序 | ORDER BY age DESC |
| groupBy | 分组 | GROUP BY age |

## 四、分页插件

### 4.1 配置分页插件

```java
package com.example.config;

import com.baomidou.mybatisplus.annotation.DbType;
import com.baomidou.mybatisplus.extension.plugins.MybatisPlusInterceptor;
import com.baomidou.mybatisplus.extension.plugins.inner.PaginationInnerInterceptor;
import org.springframework.context.annotation.Bean;
import org.springframework.context.annotation.Configuration;

@Configuration
public class MybatisPlusConfig {
    
    @Bean
    public MybatisPlusInterceptor mybatisPlusInterceptor() {
        MybatisPlusInterceptor interceptor = new MybatisPlusInterceptor();
        
        // 添加分页插件
        PaginationInnerInterceptor paginationInterceptor = new PaginationInnerInterceptor(DbType.MYSQL);
        paginationInterceptor.setOverflow(true);  // 溢出总页数后是否进行处理
        paginationInterceptor.setMaxLimit(500L);  // 单页分页条数限制
        
        interceptor.addInnerInterceptor(paginationInterceptor);
        
        return interceptor;
    }
}
```

### 4.2 使用分页

```java
@GetMapping("/page")
public IPage<User> page(
    @RequestParam(defaultValue = "1") Integer current,
    @RequestParam(defaultValue = "10") Integer size
) {
    Page<User> page = new Page<>(current, size);
    
    IPage<User> result = userService.lambdaQuery()
        .ge(User::getAge, 18)
        .page(page);
    
    return result;
}
```

**返回结果**:

```json
{
  "records": [...],      // 数据列表
  "total": 100,          // 总记录数
  "size": 10,            // 每页条数
  "current": 1,          // 当前页
  "pages": 10            // 总页数
}
```

## 五、XML映射文件

### 5.1 基本配置

**目录结构**:

```
src/main/resources/
└── mapper/
    └── UserMapper.xml
```

**UserMapper.xml**:

```xml
<?xml version="1.0" encoding="UTF-8"?>
<!DOCTYPE mapper PUBLIC "-//mybatis.org//DTD Mapper 3.0//EN" 
    "http://mybatis.org/dtd/mybatis-3-mapper.dtd">
<mapper namespace="com.example.mapper.UserMapper">
    
    <!-- 结果映射 -->
    <resultMap id="UserResultMap" type="com.example.entity.User">
        <id property="id" column="id"/>
        <result property="username" column="username"/>
        <result property="password" column="password"/>
        <result property="email" column="email"/>
        <result property="age" column="age"/>
        <result property="createTime" column="create_time"/>
        <result property="updateTime" column="update_time"/>
    </resultMap>
    
    <!-- 自定义查询 -->
    <select id="selectByUsername" resultMap="UserResultMap">
        SELECT * FROM user WHERE username = #{username}
    </select>
    
    <!-- 动态SQL -->
    <select id="selectByCondition" resultMap="UserResultMap">
        SELECT * FROM user
        <where>
            <if test="username != null and username != ''">
                AND username LIKE CONCAT('%', #{username}, '%')
            </if>
            <if test="age != null">
                AND age = #{age}
            </if>
            <if test="email != null and email != ''">
                AND email = #{email}
            </if>
        </where>
        ORDER BY create_time DESC
    </select>
    
</mapper>
```

### 5.2 动态SQL标签

#### if标签

```xml
<if test="username != null and username != ''">
    AND username = #{username}
</if>
```

#### choose-when-otherwise

```xml
<choose>
    <when test="age != null">
        AND age = #{age}
    </when>
    <when test="email != null">
        AND email = #{email}
    </when>
    <otherwise>
        AND deleted = 0
    </otherwise>
</choose>
```

#### foreach标签

```xml
<!-- IN查询 -->
<foreach collection="ids" item="id" open="(" separator="," close=")">
    #{id}
</foreach>

<!-- 批量插入 -->
<insert id="batchInsert">
    INSERT INTO user (username, password) VALUES
    <foreach collection="users" item="user" separator=",">
        (#{user.username}, #{user.password})
    </foreach>
</insert>
```

## 六、高级特性

### 6.1 自动填充

**实现MetaObjectHandler**:

```java
package com.example.handler;

import com.baomidou.mybatisplus.core.handlers.MetaObjectHandler;
import org.apache.ibatis.reflection.MetaObject;
import org.springframework.stereotype.Component;

import java.time.LocalDateTime;

@Component
public class MyMetaObjectHandler implements MetaObjectHandler {
    
    @Override
    public void insertFill(MetaObject metaObject) {
        this.strictInsertFill(metaObject, "createTime", LocalDateTime.class, LocalDateTime.now());
        this.strictInsertFill(metaObject, "updateTime", LocalDateTime.class, LocalDateTime.now());
    }
    
    @Override
    public void updateFill(MetaObject metaObject) {
        this.strictUpdateFill(metaObject, "updateTime", LocalDateTime.class, LocalDateTime.now());
    }
}
```

**实体类标注**:

```java
@TableField(fill = FieldFill.INSERT)
private LocalDateTime createTime;

@TableField(fill = FieldFill.INSERT_UPDATE)
private LocalDateTime updateTime;
```

### 6.2 乐观锁

```java
@Bean
public MybatisPlusInterceptor mybatisPlusInterceptor() {
```


### 6.3 逻辑删除

**配置**:

```yaml
mybatis-plus:
  global-config:
    db-config:
      logic-delete-field: deleted
      logic-delete-value: 1
      logic-not-delete-value: 0
```

**实体类**:

```java
@TableLogic
private Integer deleted;
```

**效果**:
- `removeById()` → 执行UPDATE设置deleted=1
- `selectList()` → 自动添加WHERE deleted=0
