## 一、路由断言工厂 (Predicate Factories)

### 1.1 什么是断言?

**定义**:
> 断言是路由匹配的条件,只有满足所有断言条件,请求才会被路由到目标服务。

**基本结构**:
```yaml
spring:
  cloud:
    gateway:
      routes:
        - id: order-service
          uri: lb://order-service
          predicates:
            - Path=/api/order/**      # 路径断言
            - Method=GET,POST         # 方法断言
            - Header=X-Request-Id, \d+ # Header断言
          filters:
            - StripPrefix=1
```

---

### 1.2 Path断言 (最常用)

#### 基础用法

```yaml
predicates:
  - Path=/api/order/**
```

**匹配规则**:
```
✅ /api/order/create       → 匹配
✅ /api/order/123          → 匹配
✅ /api/order/list?page=1  → 匹配
❌ /api/user/info          → 不匹配
❌ /order/create           → 不匹配(缺少/api前缀)
```

#### 路径变量捕获

```yaml
predicates:
  - Path=/api/order/{orderId}
```

**在过滤器中使用**:
```java
@Component
public class OrderFilter implements GlobalFilter {
    
    @Override
    public Mono<Void> filter(ServerWebExchange exchange, GatewayFilterChain chain) {
        // 获取路径变量
        String orderId = exchange.getAttribute("orderId");
        
        log.info("订单ID: {}", orderId);
        
        return chain.filter(exchange);
    }
}
```

### 1.3 Method断言

```yaml
predicates:
  - Method=GET,POST,PUT,DELETE
```

**示例**:
```yaml
# 只允许GET和POST请求
predicates:
  - Path=/api/order/**
  - Method=GET,POST

# 结果:
✅ GET /api/order/list    → 匹配
✅ POST /api/order/create → 匹配
❌ PUT /api/order/update  → 不匹配(405 Method Not Allowed)
❌ DELETE /api/order/1    → 不匹配
```

---

### 1.4 Header断言

```yaml
predicates:
  - Header=X-Request-Id, \d+  # Header名, 正则表达式
```

**示例**:
```yaml
# 要求Header中必须包含X-Request-Id且为数字
predicates:
  - Header=X-Request-Id, \d+

# 测试:
curl -H "X-Request-Id: 12345" http://localhost:8080/api/order/list
✅ 匹配

curl -H "X-Request-Id: abc" http://localhost:8080/api/order/list
❌ 不匹配(不是数字)

curl http://localhost:8080/api/order/list
❌ 不匹配(缺少Header)
```

### 1.8 Time断言

```yaml
predicates:
  - Between=2024-01-01T00:00:00.000+08:00, 2024-12-31T23:59:59.999+08:00
```

**应用场景**:
```yaml
# 活动期间路由到促销服务
routes:
  - id: promotion-service
    uri: lb://promotion-service
    predicates:
      - Path=/api/promotion/**
      - Between=2024-11-01T00:00:00.000+08:00, 2024-11-11T23:59:59.999+08:00
    filters:
      - StripPrefix=1

# 活动结束后自动失效
```

### 1.9 Weight断言 (权重路由)

```yaml

  - Weight=group1, 80  # 组名, 权重
```

**灰度发布示例**:
```yaml
routes:
  # 80%流量走旧版本
  - id: order-service-v1
    uri: lb://order-service-v1
    predicates:
      - Path=/api/order/**
      - Weight=order-group, 80
  
  # 20%流量走新版本
  - id: order-service-v2
    uri: lb://order-service-v2
    predicates:
      - Path=/api/order/**
      - Weight=order-group, 20
```

## 二、过滤器工厂 (Filter Factories)

### 2.1 过滤器类型

```
Gateway过滤器分为两类:

1. GatewayFilter (路由过滤器)
   - 作用于特定路由
   - 在filters中配置
   
2. GlobalFilter (全局过滤器)
   - 作用于所有路由
   - 需要编写Java代码
```

---

### 2.2 StripPrefix过滤器

**功能**: 去掉路径前缀

```yaml
filters:
  - StripPrefix=1
```

**示例**:
```
请求: /api/order/create
过滤后: /order/create (去掉/api)
转发到: lb://order-service/order/create
```

**多级前缀**:
```yaml
filters:
  - StripPrefix=2

请求: /api/v1/order/create
过滤后: /order/create (去掉/api/v1)
```

---

### 2.3 AddRequestHeader过滤器

**功能**: 添加请求头

```yaml
filters:
  - AddRequestHeader=X-Request-Gateway, spring-cloud-gateway
```

**示例**:
```yaml
routes:
  - id: order-service
    uri: lb://order-service
    filters:
      - AddRequestHeader=X-Request-Source, gateway
      - AddRequestHeader=X-Request-Time, #{T(java.time.LocalDateTime).now()}

# 后端服务收到的Header:
X-Request-Source: gateway
X-Request-Time: 2024-01-01T10:00:00
```

---

### 2.4 AddRequestParameter过滤器

**功能**: 添加请求参数

```yaml
filters:
  - AddRequestParameter=source, gateway
```

**示例**:
```
原始请求: /api/order/list?pageSize=10
过滤后: /api/order/list?pageSize=10&source=gateway
```

---

### 2.5 RewritePath过滤器

**功能**: 重写路径

```yaml
filters:
  - RewritePath=/api/(?<segment>.*), /$\{segment}
```

**示例**:
```yaml
routes:
  - id: order-service
    uri: lb://order-service
    predicates:
      - Path=/api/order/**
    filters:
      - RewritePath=/api/order/(?<path>.*), /order/$\{path}

# 转换:
/api/order/create → /order/create
/api/order/123    → /order/123
```

---

### 2.6 RedirectTo过滤器

**功能**: 重定向

```yaml
filters:
  - RedirectTo=302, https://www.example.com
```

**示例**:
```yaml
# 永久重定向
routes:
  - id: old-api
    uri: no://op
    predicates:
      - Path=/api/v1/**
    filters:
      - RedirectTo=301, /api/v2${request.path}
```

---

### 2.7 Retry过滤器

**功能**: 重试机制

```yaml
filters:
  - Retry=retries=3,statuses=BAD_GATEWAY
```

**完整配置**:
```yaml
filters:
  - name: Retry
    args:
      retries: 3              # 重试次数
      statuses: BAD_GATEWAY,SERVICE_UNAVAILABLE  # 触发重试的状态码
      methods: GET,POST       # 支持的方法
      backoff:
        firstBackoff: 100ms   # 首次重试间隔
        maxBackoff: 500ms     # 最大重试间隔
        factor: 2             # 退避因子
```

---

### 2.8 RequestRateLimiter过滤器 (限流)

**功能**: 请求限流

```yaml
filters:
  - name: RequestRateLimiter
    args:
      redis-rate-limiter.replenishRate: 10  # 每秒允许请求数
      redis-rate-limiter.burstCapacity: 20  # 突发容量
      redis-rate-limiter.requestedTokens: 1 # 每次请求消耗token数
```

**需要Redis支持**:
```xml
<dependency>
    <groupId>org.springframework.boot</groupId>
    <artifactId>spring-boot-starter-data-redis-reactive</artifactId>
</dependency>
```

**配置Redis**:
```yaml
spring:
  redis:
    host: localhost
    port: 6379
```

## 三、全局过滤器 (GlobalFilter)

### 3.1 自定义全局过滤器

#### 示例1: 日志记录过滤器

```java
@Component
@Slf4j
public class LoggingGlobalFilter implements GlobalFilter, Ordered {
    
    @Override
    public Mono<Void> filter(ServerWebExchange exchange, GatewayFilterChain chain) {
        ServerHttpRequest request = exchange.getRequest();
        
        // 记录请求信息
        log.info("请求路径: {}", request.getPath());
        log.info("请求方法: {}", request.getMethod());
        log.info("请求IP: {}", request.getRemoteAddress());
        log.info("请求时间: {}", LocalDateTime.now());
        
        // 记录响应时间
        long startTime = System.currentTimeMillis();
        
        return chain.filter(exchange).then(Mono.fromRunnable(() -> {
            long endTime = System.currentTimeMillis();
            log.info("响应时间: {}ms", endTime - startTime);
        }));
    }
    
    @Override
    public int getOrder() {
        return -1; // 优先级,数值越小越先执行
    }
}
```

#### 示例2: JWT鉴权过滤器

```java
@Component
public class AuthGlobalFilter implements GlobalFilter, Ordered {
    
    @Autowired
    private JwtUtil jwtUtil;
    
    // 白名单(不需要认证的路径)
    private static final List<String> WHITE_LIST = Arrays.asList(
        "/api/auth/login",
        "/api/auth/register",
        "/api/public/**"
    );
    
    @Override
    public Mono<Void> filter(ServerWebExchange exchange, GatewayFilterChain chain) {
        ServerHttpRequest request = exchange.getRequest();
        String path = request.getPath().toString();
        
        // 检查是否在白名单中
        if (isWhiteList(path)) {
            return chain.filter(exchange);
        }
        
        // 获取Token
        String token = request.getHeaders().getFirst("Authorization");
        
        if (StringUtils.isBlank(token)) {
            return unauthorized(exchange, "缺少认证Token");
        }
        
        // 验证Token
        try {
            Claims claims = jwtUtil.parseToken(token);
            
            // 将用户信息传递给下游服务
            ServerHttpRequest mutatedRequest = request.mutate()
                .header("X-User-Id", claims.getSubject())
                .header("X-User-Name", claims.get("userName", String.class))
                .build();
            
            ServerWebExchange mutatedExchange = exchange.mutate()
                .request(mutatedRequest)
                .build();
            
            return chain.filter(mutatedExchange);
            
        } catch (Exception e) {
            return unauthorized(exchange, "Token无效或已过期");
        }
    }
    
    private boolean isWhiteList(String path) {
        return WHITE_LIST.stream()
            .anyMatch(whitePath -> AntPathMatcher.INSTANCE.match(whitePath, path));
    }
    
    private Mono<Void> unauthorized(ServerWebExchange exchange, String message) {
        exchange.getResponse().setStatusCode(HttpStatus.UNAUTHORIZED);
        exchange.getResponse().getHeaders().setContentType(MediaType.APPLICATION_JSON);
        
        Result<?> result = Result.error(401, message);
        String body = JSON.toJSONString(result);
        
        DataBuffer buffer = exchange.getResponse()
            .bufferFactory()
            .wrap(body.getBytes(StandardCharsets.UTF_8));
        
        return exchange.getResponse().writeWith(Mono.just(buffer));
    }
    
    @Override
    public int getOrder() {
        return 0;
    }
}
```

#### 示例3: 跨域处理过滤器

```java
@Configuration
public class CorsConfig {
    
    @Bean
    public CorsWebFilter corsWebFilter() {
        CorsConfiguration config = new CorsConfiguration();
        
        // 允许的源
        config.addAllowedOrigin("*");
        
        // 允许的方法
        config.addAllowedMethod("GET");
        config.addAllowedMethod("POST");
        config.addAllowedMethod("PUT");
        config.addAllowedMethod("DELETE");
        
        // 允许的Header
        config.addAllowedHeader("*");
        
        // 暴露的Header
        config.addExposedHeader("X-Custom-Header");
        
        // 预检请求缓存时间
        config.setMaxAge(3600L);
        
        UrlBasedCorsConfigurationSource source = new UrlBasedCorsConfigurationSource();
        source.registerCorsConfiguration("/**", config);
        
        return new CorsWebFilter(source);
    }
}
```

### 3.2 过滤器执行顺序

```
请求进入Gateway
    ↓
┌─────────────────────┐
│  GlobalFilter (-1)   │ ← 最先执行(数值最小)
└─────────┬───────────┘
          ↓
┌─────────────────────┐
│  GlobalFilter (0)    │
└─────────┬───────────┘
          ↓
┌─────────────────────┐
│  Route Filter        │ ← 路由过滤器
└─────────┬───────────┘
          ↓
┌─────────────────────┐
│  GlobalFilter (100)  │ ← 最后执行(数值最大)
└─────────┬───────────┘
          ↓
转发到下游服务
```

**控制顺序**:
```java
@Component
public class FirstFilter implements GlobalFilter, Ordered {
    @Override
    public int getOrder() {
        return -100; // 非常早执行
    }
}

@Component
public class LastFilter implements GlobalFilter, Ordered {
    @Override
    public int getOrder() {
        return 100; // 非常晚执行
    }
}
```

```java
@Component
public class DynamicRouteService implements ApplicationEventPublisherAware {
    
    @Autowired
    private RouteDefinitionWriter routeDefinitionWriter;
    
    private ApplicationEventPublisher publisher;
    
    /**
     * 更新路由
     */
    public void updateRoute(RouteDefinition definition) {
        routeDefinitionWriter.save(Mono.just(definition)).subscribe();
        this.publisher.publishEvent(new RefreshRoutesEvent(this));
    }
    
    /**
     * 删除路由
     */
    public void deleteRoute(String routeId) {
        routeDefinitionWriter.delete(Mono.just(routeId)).subscribe();
        this.publisher.publishEvent(new RefreshRoutesEvent(this));
    }
    
    @Override
    public void setApplicationEventPublisher(ApplicationEventPublisher publisher) {
        this.publisher = publisher;
    }
}
```

### 4.2 数据库动态路由

#### 表结构设计

```sql
CREATE TABLE gateway_route (
    id BIGINT PRIMARY KEY AUTO_INCREMENT,
    route_id VARCHAR(64) NOT NULL COMMENT '路由ID',
    uri VARCHAR(255) NOT NULL COMMENT '目标URI',
    predicates TEXT COMMENT '断言配置(JSON)',
    filters TEXT COMMENT '过滤器配置(JSON)',
    enabled TINYINT DEFAULT 1 COMMENT '是否启用',
    create_time DATETIME DEFAULT CURRENT_TIMESTAMP,
    update_time DATETIME DEFAULT CURRENT_TIMESTAMP ON UPDATE CURRENT_TIMESTAMP,
    INDEX idx_route_id (route_id)
);
```

#### 实体类

```java
@Data
@TableName("gateway_route")
public class GatewayRoute {
    
    @TableId(type = IdType.AUTO)
    private Long id;
    
    private String routeId;
    
    private String uri;
    
    private String predicates;
    
    private String filters;
    
    private Integer enabled;
}
```

#### 加载路由

```java
@Component
public class DatabaseRouteLoader implements ApplicationRunner {
    
    @Autowired
    private GatewayRouteMapper routeMapper;
    
    @Autowired
    private RouteDefinitionWriter routeDefinitionWriter;
    
    @Override
    public void run(ApplicationArguments args) {
        // 从数据库加载路由
        List<GatewayRoute> routes = routeMapper.selectList(
            new LambdaQueryWrapper<GatewayRoute>()
                .eq(GatewayRoute::getEnabled, 1)
        );
        
        for (GatewayRoute route : routes) {
            RouteDefinition definition = new RouteDefinition();
            definition.setId(route.getRouteId());
            definition.setUri(URI.create(route.getUri()));
            
            // 解析断言
            List<PredicateDefinition> predicates = JSON.parseArray(
                route.getPredicates(), PredicateDefinition.class
            );
            definition.setPredicates(predicates);
            
            // 解析过滤器
            List<FilterDefinition> filters = JSON.parseArray(
                route.getFilters(), FilterDefinition.class
            );
            definition.setFilters(filters);
            
            // 保存路由
            routeDefinitionWriter.save(Mono.just(definition)).subscribe();
        }
        
        log.info("从数据库加载了{}条路由", routes.size());
    }
}
```

## 五、负载均衡策略

### 5.1 内置负载均衡器

```yaml
uri格式:
lb://service-name  # 使用LoadBalancer

示例:
uri: lb://order-service
```

**支持的策略**:
```
- RoundRobin (轮询,默认)
- Random (随机)
- WeightedResponseTime (加权响应时间)
```

### 5.2 自定义负载均衡策略

```java
@Configuration
public class LoadBalancerConfig {
    
    @Bean
    public ReactorLoadBalancer<ServiceInstance> randomLoadBalancer(
            Environment environment,
            LoadBalancerClientFactory loadBalancerClientFactory) {
        
        String name = environment.getProperty(LoadBalancerClientFactory.PROPERTY_NAME);
        
        return new RandomLoadBalancer(
            loadBalancerClientFactory.getLazyProvider(name, ServiceInstanceListSupplier.class),
            name
        );
    }
}
```

**在路由中使用**:
```yaml
routes:
  - id: order-service
    uri: lb://order-service
    metadata:
      load-balancer: random  # 使用随机策略
```

### 6.3 缓存优化

```java
@Component
public class ResponseCacheFilter implements GlobalFilter {
    
    private final Cache<String, String> cache = Caffeine.newBuilder()
        .maximumSize(1000)
        .expireAfterWrite(5, TimeUnit.MINUTES)
        .build();
    
    @Override
    public Mono<Void> filter(ServerWebExchange exchange, GatewayFilterChain chain) {
        String key = exchange.getRequest().getPath().toString();
        
        // 只缓存GET请求
        if (!exchange.getRequest().getMethod().equals(HttpMethod.GET)) {
            return chain.filter(exchange);
        }
        
        // 尝试从缓存获取
        String cachedBody = cache.getIfPresent(key);
        if (cachedBody != null) {
            DataBuffer buffer = exchange.getResponse()
                .bufferFactory()
                .wrap(cachedBody.getBytes());
            
            exchange.getResponse().getHeaders()
                .setContentType(MediaType.APPLICATION_JSON);
            exchange.getResponse().getHeaders()
                .add("X-Cache", "HIT");
            
            return exchange.getResponse().writeWith(Mono.just(buffer));
        }
        
        // 缓存未命中,继续执行
        return chain.filter(exchange).then(Mono.fromRunnable(() -> {
            // 将响应写入缓存
            // ...
        }));
    }
}
```

```
原则:
1. 尽早失败(鉴权放在前面)
2. 减少不必要的过滤
3. 避免阻塞操作
```

### 8.3 错误处理

```java
@Component
public class ErrorHandlingFilter implements GlobalFilter {
    
    @Override
    public Mono<Void> filter(ServerWebExchange exchange, GatewayFilterChain chain) {
        return chain.filter(exchange).onErrorResume(e -> {
            log.error("Gateway异常", e);
            
            exchange.getResponse().setStatusCode(HttpStatus.BAD_GATEWAY);
            
            Result<?> result = Result.error(502, "网关异常: " + e.getMessage());
            String body = JSON.toJSONString(result);
            
            DataBuffer buffer = exchange.getResponse()
                .bufferFactory()
                .wrap(body.getBytes());
            
            return exchange.getResponse().writeWith(Mono.just(buffer));
        });
    }
}
```
