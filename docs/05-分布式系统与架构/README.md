## 📚 模块概述

本模块系统地整理了分布式系统和微服务架构的核心技术，包括：

- **配置中心**：Spring Cloud Config、Apollo、Nacos
- **分布式事务**：Seata、Atomikos、LCN、TCC

## 🎯 学习路径

### 📘 初级阶段（1个月）

**目标**：理解分布式系统基本概念

1. **微服务基础**
   - 微服务架构概述
   - 服务拆分原则
   - RESTful API 设计

2. **服务通信**
   - HTTP/REST
   - RPC
   - 消息队列

### 📗 中级阶段（2-3个月）

**目标**：掌握微服务治理核心组件

3. **服务注册与发现**
   - Eureka
   - Nacos
   - Consul

4. **API 网关**
   - 网关作用
   - Spring Cloud Gateway
   - 路由和过滤

5. **配置管理**
   - 配置中心选型
   - Apollo 使用
   - 动态刷新

### 📙 高级阶段（3-4个月）

**目标**：掌握分布式系统高级特性

6. **流量控制与容错**
   - 限流算法
   - 熔断降级
   - Sentinel/Hystrix

7. **分布式事务**
   - CAP 理论
   - BASE 理论
   - Seata 实战

8. **分布式一致性**
   - Raft 算法
   - Paxos 算法
   - ZooKeeper

## 💡 学习建议

- 先理解分布式系统的挑战

- 学会性能调优
- 建立监控体系

## 🛠️ 常用工具

### 服务治理
- **Spring Cloud**：微服务全家桶
- **Dubbo**：阿里 RPC 框架
- **gRPC**：Google RPC 框架

### 注册中心
- **Nacos**：阿里服务发现
- **Eureka**：Netflix 注册中心
- **Consul**：HashiCorp 服务网格

### API 网关
- **Spring Cloud Gateway**：Spring 官方网关
- **Kong**：基于 Nginx 的网关
- **Zuul**：Netflix 网关

### 配置中心
- **Apollo**：携程配置中心
- **Nacos**：阿里配置中心
- **Consul**：HashiCorp KV 存储

### 分布式事务
- **Seata**：阿里分布式事务
- **Atomikos**：JTA 实现
- **RocketMQ**：消息事务
