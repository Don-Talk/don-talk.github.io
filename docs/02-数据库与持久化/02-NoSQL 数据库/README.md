## 🎯 学习路径

### Redis 学习路线

1. **基础阶段**
   - 五大数据结构（String、List、Set、Hash、ZSet）
   - 常用命令
   - 持久化机制（RDB、AOF）

2. **进阶阶段**
   - 主从复制
   - 哨兵模式
   - Cluster 集群

3. **高级阶段**
   - 缓存设计模式
   - 缓存穿透/击穿/雪崩解决方案
   - 分布式锁实现

### MongoDB 学习路线

1. **基础概念**
   - 文档模型
   - 集合与数据库
   - BSON 格式

2. **CRUD 操作**
   - 插入文档
   - 查询文档
   - 更新文档
   - 删除文档

## 💡 应用场景

### Redis
- ✅ 缓存系统
- ✅ 会话存储
- ✅ 计数器
- ✅ 排行榜
- ✅ 消息队列
- ✅ 分布式锁

### MongoDB
- ✅ 内容管理系统
- ✅ 日志存储
- ✅ 实时分析
- ✅ 物联网数据
- ✅ 移动应用后端
