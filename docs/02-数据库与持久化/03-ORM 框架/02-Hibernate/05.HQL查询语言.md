## 1. HQL 简介

### 1.1 什么是 HQL？

HQL（Hibernate Query Language）是 Hibernate 的面向对象查询语言，类似于 SQL，但操作的是对象而非表。

**HQL vs SQL**：

| 特性 | HQL | SQL |
|------|-----|-----|
| **操作对象** | 类和属性 | 表和列 |
| **大小写敏感** | 类名和属性名敏感，关键字不敏感 | 取决于数据库 |
| **可移植性** | 跨数据库 | 依赖数据库 |
| **示例** | `FROM User WHERE age > 20` | `SELECT * FROM user WHERE age > 20` |

### 1.2 基本语法

```java
// SQL
SELECT * FROM user WHERE age > 20;

// HQL
FROM User WHERE age > 20
```

## 2. 基础查询

### 2.1 查询所有

```java
@Test
public void testFindAll() {
    Session session = sessionFactory.openSession();
    
    // 方式一：完整写法
    List<User> users = session.createQuery("FROM User", User.class).list();
    
    // 方式二：简写
    List<User> users2 = session.createQuery("FROM com.example.entity.User", User.class).list();
    
    users.forEach(System.out::println);
    
    session.close();
}
```

### 2.2 选择特定字段

```java
@Test
public void testSelectFields() {
    Session session = sessionFactory.openSession();
    
    // 查询单个字段
    List<String> usernames = session.createQuery(
            "SELECT username FROM User", String.class)
            .list();
    
    // 查询多个字段（返回 Object[]）
    List<Object[]> results = session.createQuery(
            "SELECT username, email FROM User")
            .list();
    
    for (Object[] row : results) {
        System.out.println("用户名: " + row[0] + ", 邮箱: " + row[1]);
    }
    
    session.close();
}
```

### 2.3 使用构造器

## 3. 条件查询

### 3.1 WHERE 子句

```java
@Test
public void testWhereClause() {
    Session session = sessionFactory.openSession();
    
    // 等于
    List<User> users = session.createQuery(
            "FROM User WHERE age = 25", User.class)
            .list();
    
    // 不等于
    List<User> users2 = session.createQuery(
            "FROM User WHERE age <> 25", User.class)
            .list();
    
    session.close();
}
```

### 3.2 比较运算符

```java
@Test
public void testComparison() {
    Session session = sessionFactory.openSession();
    
    // 大于、小于

            "FROM User WHERE age > 20 AND age < 30", User.class)
            .list();
    
    // 大于等于、小于等于
    List<User> users2 = session.createQuery(
            "FROM User WHERE age >= 18 AND age <= 60", User.class)
            .list();
    
    session.close();
}
```

### 3.3 逻辑运算符

### 3.5 IN

```java
@Test
public void testIn() {

    // 固定值
```

## 6. 分页查询

### 6.1 setFirstResult 和 setMaxResults

```java
@Test
public void testPagination() {
    Session session = sessionFactory.openSession();
    
    int pageNum = 1;      // 第几页（从 1 开始）
    int pageSize = 10;    // 每页条数
    
    List<User> users = session.createQuery("FROM User", User.class)
            .setFirstResult((pageNum - 1) * pageSize)  // 起始位置
            .setMaxResults(pageSize)                    // 最大结果数
            .list();
    
    System.out.println("第 " + pageNum + " 页，共 " + users.size() + " 条");
    
    session.close();
}
```

### 6.2 封装分页工具

```java
public class PageHelper {
    
    public static <T> List<T> findPage(Session session, String hql, 
                                        Class<T> clazz, int pageNum, int pageSize) {
        return session.createQuery(hql, clazz)
                .setFirstResult((pageNum - 1) * pageSize)
                .setMaxResults(pageSize)
                .list();
    }
    
    public static Long count(Session session, String hql) {
        return session.createQuery("SELECT COUNT(*) " + hql, Long.class)
                .uniqueResult();
    }
}

// 使用
List<User> users = PageHelper.findPage(session, "FROM User", User.class, 1, 10);
Long total = PageHelper.count(session, "FROM User");
```

## 7. 聚合函数

### 7.1 常用聚合函数

```java
@Test
public void testAggregation() {
    Session session = sessionFactory.openSession();
    
    // COUNT
    Long count = session.createQuery("SELECT COUNT(*) FROM User", Long.class)
            .uniqueResult();

            .uniqueResult();
    System.out.println("最大年龄: " + maxAge);
    
    // MIN

    Long sumAge = session.createQuery("SELECT SUM(age) FROM User", Long.class)
            .uniqueResult();
    System.out.println("年龄总和: " + sumAge);
    
    session.close();
}
```

## 8. 关联查询

### 8.1 JOIN 类型

* **INNER JOIN**：内连接
* **LEFT JOIN**：左外连接
* **RIGHT JOIN**：右外连接
* **FULL JOIN**：全外连接（部分数据库不支持）

### 8.2 INNER JOIN

```java
@Test
public void testInnerJoin() {
    Session session = sessionFactory.openSession();
    
    // 查询有订单的用户
    List<User> users = session.createQuery(
            "SELECT DISTINCT u FROM User u INNER JOIN u.orders o", User.class)
            .list();
    
    session.close();
}
```

### 8.3 LEFT JOIN

```java
@Test
public void testLeftJoin() {
    Session session = sessionFactory.openSession();
    
    // 查询所有用户及其订单（包括没有订单的用户）
    List<Object[]> results = session.createQuery(
            "SELECT u, o FROM User u LEFT JOIN u.orders o")
            .list();
    
    for (Object[] row : results) {
        User user = (User) row[0];
        Order order = (Order) row[1];
        System.out.println("用户: " + user.getUsername() + 
                          ", 订单: " + (order != null ? order.getOrderNo() : "无"));
    }
    
    session.close();
}
```

### 8.4 JOIN FETCH（ eagerly 加载）

```java
@Test
public void testJoinFetch() {
    Session session = sessionFactory.openSession();
    
    // 一次性加载用户和订单，避免 N+1 问题
    List<User> users = session.createQuery(
            "FROM User u LEFT JOIN FETCH u.orders", User.class)
            .list();
    
    for (User user : users) {
        System.out.println("用户: " + user.getUsername());
        System.out.println("订单数: " + user.getOrders().size());
    }
    
    session.close();
}
```

## 10. 命名查询

### 10.1 @NamedQuery

```java
@Entity
@NamedQuery(
    name = "User.findByUsername",
    query = "FROM User WHERE username = :username"
)
@NamedQuery(
    name = "User.findByAgeRange",
    query = "FROM User WHERE age BETWEEN :minAge AND :maxAge"
)
public class User {
    // ...
}
```

**使用**：

```java
@Test
public void testNamedQuery() {
    Session session = sessionFactory.openSession();
    
    User user = session.createNamedQuery("User.findByUsername", User.class)
            .setParameter("username", "张三")
            .uniqueResult();
    
    List<User> users = session.createNamedQuery("User.findByAgeRange", User.class)
            .setParameter("minAge", 20)
            .setParameter("maxAge", 30)
            .list();
    
    session.close();
}
```

### 10.2 @NamedQueries

```java
@Entity
@NamedQueries({
    @NamedQuery(name = "User.findAll", query = "FROM User"),
    @NamedQuery(name = "User.findByUsername", query = "FROM User WHERE username = :username"),
    @NamedQuery(name = "User.deleteOldUsers", query = "DELETE FROM User WHERE age < :age")
})
public class User {
    // ...
}
```

### 10.3 XML 命名查询

在 `User.hbm.xml` 中：

```xml
<hibernate-mapping>
    <class name="com.example.entity.User" table="user">
        <!-- 映射配置 -->
    </class>
    
    <query name="User.findByUsername">
        <![CDATA[
            FROM User WHERE username = :username
        ]]>
    </query>
    
    <sql-query name="User.findRawData">
        <![CDATA[
            SELECT * FROM user WHERE username = :username
        ]]>
    </sql-query>
</hibernate-mapping>
```

### 12.2 统计查询

```java
@Test
public void testStatistics() {
    Session session = sessionFactory.openSession();
    
    // 按性别和年龄段统计
    List<Object[]> results = session.createQuery(
            "SELECT gender, " +
```

## 14. 总结

### 14.1 核心知识点

✅ **基础查询**：SELECT、FROM、WHERE  
✅ **条件查询**：比较、逻辑、LIKE、IN、BETWEEN  
✅ **参数绑定**：位置参数、命名参数  
✅ **排序分页**：ORDER BY、setFirstResult、setMaxResults  
✅ **聚合函数**：COUNT、SUM、AVG、MAX、MIN  
✅ **关联查询**：JOIN、JOIN FETCH  
✅ **子查询**：WHERE 子句、EXISTS、IN  
✅ **命名查询**：@NamedQuery、XML  
✅ **原生 SQL**：createNativeQuery  

### 14.2 最佳实践

📌 优先使用命名参数而非位置参数  
📌 使用 JOIN FETCH 解决 N+1 问题  
📌 大量数据时使用分页查询  
📌 常用查询使用命名查询  
📌 复杂查询考虑使用原生 SQL
