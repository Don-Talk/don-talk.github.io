## 1. Spring Data JPA 简介

### 1.1 什么是 Spring Data JPA？

Spring Data JPA 是 Spring 基于 Hibernate 开发的 JPA 框架，提供了 Repository 抽象，极大简化了数据访问层的开发。

**优势**：
* ✅ 无需编写实现类
* ✅ 方法名即查询
* ✅ 内置 CRUD 方法
* ✅ 支持分页、排序
* ✅ 自动审计功能

### 1.2 技术栈关系

```
Spring Data JPA（简化开发）
    ↓
JPA（规范/接口）
    ↓
Hibernate（实现）
    ↓
数据库
```

## 3. JpaRepository 核心方法

### 3.1 保存操作

```java
@Service
public class UserService {
    
    @Autowired
    private UserRepository userRepository;
    
    // 保存单个
    public User save(User user) {
        return userRepository.save(user);
    }

    public List<User> saveAll(List<User> users) {
        return userRepository.saveAll(users);
    }
```

### 3.2 查询操作

```java
// 查询所有
List<User> users = userRepository.findAll();

// 根据 ID 查询
User user = userRepository.findById(1L).orElse(null);

// 判断是否存在
boolean exists = userRepository.existsById(1L);

// 统计总数
long count = userRepository.count();
```

### 3.3 删除操作

```java
// 根据 ID 删除
userRepository.deleteById(1L);

// 删除对象
userRepository.delete(user);

userRepository.deleteAll();
```

### 3.4 分页排序

```java

Page<User> page = userRepository.findAll(PageRequest.of(0, 10));
```

## 4. 方法名派生查询

### 4.1 命名规则

Spring Data JPA 会根据方法名自动生成查询语句。

**格式**：`find|read|get|query|count|delete + By + 条件`

### 4.2 常用关键字

| 关键字 | 示例 | SQL |
|--------|------|-----|
| And | findByUsernameAndAge | WHERE username = ? AND age = ? |
| Or | findByUsernameOrEmail | WHERE username = ? OR email = ? |
| Between | findByAgeBetween | WHERE age BETWEEN ? AND ? |
| LessThan | findByAgeLessThan | WHERE age < ? |
| GreaterThan | findByAgeGreaterThan | WHERE age > ? |
| Like | findByUsernameLike | WHERE username LIKE ? |
| StartsWith | findByUsernameStartingWith | WHERE username LIKE '?%' |
| EndsWith | findByUsernameEndingWith | WHERE username LIKE '%?' |
| Contains | findByUsernameContaining | WHERE username LIKE '%?%' |
| IsNull | findByEmailIsNull | WHERE email IS NULL |
| NotNull | findByEmailNotNull | WHERE email IS NOT NULL |
| In | findByAgeIn | WHERE age IN (?) |
| NotIn | findByAgeNotIn | WHERE age NOT IN (?) |
| True | findByActiveTrue | WHERE active = true |
| False | findByActiveFalse | WHERE active = false |
| OrderBy | findByAgeOrderByUsernameDesc | ORDER BY username DESC |

### 4.3 示例

```java
public interface UserRepository extends JpaRepository<User, Long> {
    
    // 根据用户名查询
    User findByUsername(String username);
    
    // 根据年龄范围查询
    List<User> findByAgeBetween(int minAge, int maxAge);
    
    // 模糊查询
    List<User> findByUsernameContaining(String keyword);
    
    // 多条件查询
    List<User> findByAgeGreaterThanAndEmailNotNull(int age);
    
    // 排序
    List<User> findByAgeOrderByUsernameDesc(int age);
    
    // 统计
    long countByEmailContains(String domain);
    
    // 删除
    void deleteByUsername(String username);
}
```

## 5. @Query 自定义查询

### 5.1 JPQL 查询

```java
public interface UserRepository extends JpaRepository<User, Long> {
    
    // 简单查询
    @Query("FROM User WHERE age > :age")
    List<User> findUsersOlderThan(@Param("age") int age);
    
    // 多条件查询
    @Query("FROM User WHERE username LIKE %:keyword% AND age BETWEEN :minAge AND :maxAge")
    List<User> searchUsers(@Param("keyword") String keyword,
                          @Param("minAge") int minAge,
                          @Param("maxAge") int maxAge);
    
    // 返回单个字段
    @Query("SELECT u.username FROM User u WHERE u.id = :id")
    String findUsernameById(@Param("id") Long id);
    
    // 更新操作
    @Modifying
    @Transactional
    @Query("UPDATE User SET username = :username WHERE id = :id")
    int updateUsername(@Param("id") Long id, @Param("username") String username);
    
    // 删除操作
    @Modifying
    @Transactional
    @Query("DELETE FROM User WHERE age < :age")
    int deleteUsersYoungerThan(@Param("age") int age);
}
```

### 5.2 原生 SQL 查询

```java
public interface UserRepository extends JpaRepository<User, Long> {
    
    @Query(value = "SELECT * FROM user WHERE age > :age", nativeQuery = true)
    List<User> findUsersOlderThan(@Param("age") int age);
    
    @Query(value = "SELECT COUNT(*) FROM user WHERE email LIKE %:domain", 
           nativeQuery = true)
    long countByEmailDomain(@Param("domain") String domain);
}
```

## 6. Specification 动态查询

### 6.1 创建 Specification

```java
public class UserSpecification {
    
    public static Specification<User> hasUsername(String username) {
        return (root, query, cb) -> {
            if (username == null || username.isEmpty()) {
                return null;
            }
            return cb.like(root.get("username"), "%" + username + "%");
        };
    }
    
    public static Specification<User> hasAgeRange(Integer minAge, Integer maxAge) {
        return (root, query, cb) -> {
            if (minAge != null && maxAge != null) {
                return cb.between(root.get("age"), minAge, maxAge);
            } else if (minAge != null) {
                return cb.greaterThanOrEqualTo(root.get("age"), minAge);
            } else if (maxAge != null) {
                return cb.lessThanOrEqualTo(root.get("age"), maxAge);
            }
            return null;
        };
    }
    
    public static Specification<User> hasEmailNotNull() {
        return (root, query, cb) -> cb.isNotNull(root.get("email"));
    }
}
```

### 6.2 Repository 支持

```java
public interface UserRepository extends JpaRepository<User, Long>, 
                                       JpaSpecificationExecutor<User> {
    // 继承 JpaSpecificationExecutor 以支持 Specification
}
```

### 6.3 使用 Specification

```java
@Service
public class UserService {
    
    @Autowired
    private UserRepository userRepository;
    
    public List<User> search(String username, Integer minAge, Integer maxAge) {
        Specification<User> spec = Specification.where(null);
        
        if (username != null) {
            spec = spec.and(UserSpecification.hasUsername(username));
        }
        
        if (minAge != null || maxAge != null) {
            spec = spec.and(UserSpecification.hasAgeRange(minAge, maxAge));
        }
        
        spec = spec.and(UserSpecification.hasEmailNotNull());
        
        return userRepository.findAll(spec);
    }
    
    // 分页查询
    public Page<User> searchPage(String username, int pageNum, int pageSize) {
        Specification<User> spec = UserSpecification.hasUsername(username);
        PageRequest pageRequest = PageRequest.of(pageNum, pageSize);
        
        return userRepository.findAll(spec, pageRequest);
    }
}
```

## 7. 事务管理

### 7.1 声明式事务

```java
@Service
@Transactional
public class UserService {
    
    @Autowired
    private UserRepository userRepository;
    
    @Autowired
    private OrderRepository orderRepository;
    
    public void createUserWithOrder(User user, Order order) {
        userRepository.save(user);
        order.setUser(user);
        orderRepository.save(order);
        
        // 如果抛出异常，自动回滚
    }
}
```

## 9. 完整示例

### 9.1 Controller 层

```java
@RestController
@RequestMapping("/api/users")
public class UserController {
    
    @Autowired
    private UserService userService;
    
    @GetMapping
    public Page<User> list(
            @RequestParam(required = false) String username,
            @RequestParam(defaultValue = "0") int page,
            @RequestParam(defaultValue = "10") int size) {
        return userService.searchPage(username, page, size);
    }
    
    @GetMapping("/{id}")
    public User get(@PathVariable Long id) {
        return userService.findById(id);
    }
    
    @PostMapping
    public User create(@RequestBody User user) {
        return userService.save(user);
    }
    
    @PutMapping("/{id}")
    public User update(@PathVariable Long id, @RequestBody User user) {
        user.setId(id);
        return userService.update(user);
    }
    
    @DeleteMapping("/{id}")
    public void delete(@PathVariable Long id) {
        userService.delete(id);
    }
}
```

### 9.2 Service 层

```java
@Service
@Transactional
public class UserService {
    
    @Autowired
    private UserRepository userRepository;
    
    @Transactional(readOnly = true)
    public User findById(Long id) {
        return userRepository.findById(id)
                .orElseThrow(() -> new RuntimeException("用户不存在"));
    }
    
    public User save(User user) {
        return userRepository.save(user);
    }
    
    public User update(User user) {
        User existing = findById(user.getId());
        existing.setUsername(user.getUsername());
        existing.setEmail(user.getEmail());
        existing.setAge(user.getAge());
        return userRepository.save(existing);
    }
    
    public void delete(Long id) {
        userRepository.deleteById(id);
    }
    
    @Transactional(readOnly = true)
    public Page<User> searchPage(String username, int page, int size) {
        Specification<User> spec = Specification.where(null);
        
        if (username != null && !username.isEmpty()) {
            spec = spec.and(UserSpecification.hasUsername(username));
        }
        
        PageRequest pageRequest = PageRequest.of(page, size, 
                Sort.by("createTime").descending());
        
        return userRepository.findAll(spec, pageRequest);
    }
}
```

## 10. 总结

### 10.1 核心知识点

✅ **JpaRepository**：内置 CRUD 方法  
✅ **方法名派生**：findBy、countBy、deleteBy  
✅ **@Query**：JPQL 和原生 SQL  
✅ **Specification**：动态条件查询  
✅ **事务管理**：@Transactional  
✅ **审计功能**：@CreatedDate、@LastModifiedDate  

### 10.2 最佳实践

📌 优先使用方法名派生查询  
📌 复杂查询使用 @Query  
📌 动态条件使用 Specification  
📌 查询方法加 @Transactional(readOnly = true)  
📌 启用审计功能自动填充时间字段
