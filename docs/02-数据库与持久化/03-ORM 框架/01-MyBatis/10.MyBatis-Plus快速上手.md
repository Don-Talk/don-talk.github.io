## 1. MyBatis-Plus 简介

### 1.1 什么是 MyBatis-Plus？

MyBatis-Plus（简称 MP）是一个 MyBatis 的增强工具，在 MyBatis 的基础上只做增强不做改变，为简化开发、提高效率而生。

* **官网**：https://baomidou.com/
* **GitHub**：https://github.com/baomidou/mybatis-plus
* **当前版本**：3.5.x

### 1.2 核心特性

✅ **无侵入**：只做增强不做改变  
✅ **损耗小**：启动自动注入 CRUD，性能无损耗  
✅ **强大的 CRUD 操作**：内置通用 Mapper、Service  
✅ **支持 Lambda 形式调用**：通过 Lambda 表达式编写查询条件  
✅ **支持主键自动生成**：支持多种主键策略  
✅ **内置分页插件**：无需配置即可使用  
✅ **内置性能分析插件**：可输出 SQL 语句及其执行时间  

### 1.3 MyBatis vs MyBatis-Plus 对比

| 特性 | MyBatis | MyBatis-Plus |
|------|---------|--------------|
| **CRUD** | 需要手写 SQL | 内置通用 CRUD |
| **分页** | 需要配置插件 | 内置分页插件 |
| **条件构造** | 手动拼接 | Wrapper 条件构造器 |
| **代码生成** | 无 | 内置代码生成器 |
| **学习成本** | 中等 | 低 |
| **灵活性** | 高 | 高（可手写 SQL） |

### 2.4 实体类

```java
package com.example.entity;

import com.baomidou.mybatisplus.annotation.*;
import lombok.Data;
import java.io.Serializable;
import java.time.LocalDateTime;

@Data
@TableName("user")  // 指定表名
public class User implements Serializable {
    
    private static final long serialVersionUID = 1L;
    
    @TableId(type = IdType.AUTO)  // 主键自增
    private Long id;
    
    @TableField("username")  // 字段名（可省略，默认驼峰转下划线）
    private String username;
    
    private String email;
    
    private Integer age;
    
    private Integer gender;
    
    @TableLogic  // 逻辑删除
    private Integer deleted;
    
    @TableField(fill = FieldFill.INSERT)  // 插入时自动填充
    private LocalDateTime createTime;
    
    @TableField(fill = FieldFill.INSERT_UPDATE)  // 插入和更新时自动填充
    private LocalDateTime updateTime;
}
```

## 4. 条件构造器（Wrapper）

### 4.1 QueryWrapper（查询条件）

```java
@Test
public void testQueryWrapper() {
    QueryWrapper<User> wrapper = new QueryWrapper<>();
    
    // WHERE age > 20 AND age < 30
    wrapper.gt("age", 20)
           .lt("age", 30);
    
    // WHERE username LIKE '%张%'
    wrapper.like("username", "张");
    
    // WHERE age IN (20, 25, 30)
    wrapper.in("age", 20, 25, 30);
    
    // ORDER BY age DESC
    wrapper.orderByDesc("age");
    
    List<User> users = userMapper.selectList(wrapper);
    users.forEach(System.out::println);
}
```

### 4.2 LambdaQueryWrapper（推荐）

```java
@Test
public void testLambdaQueryWrapper() {
    LambdaQueryWrapper<User> wrapper = new LambdaQueryWrapper<>();
    
    // WHERE age BETWEEN 20 AND 30
    wrapper.between(User::getAge, 20, 30);
    
    // AND username LIKE '%张%'
    wrapper.like(User::getUsername, "张");
    
    // AND gender = 1
    wrapper.eq(User::getGender, 1);
    
    // ORDER BY create_time DESC
    wrapper.orderByDesc(User::getCreateTime);
    
    List<User> users = userMapper.selectList(wrapper);
    users.forEach(System.out::println);
}
```

### 4.3 常用条件方法

| 方法 | 说明 | 示例 |
|------|------|------|
| `eq` | 等于 = | `eq(User::getAge, 25)` |
| `ne` | 不等于 <> | `ne(User::getStatus, 0)` |
| `gt` | 大于 > | `gt(User::getAge, 18)` |
| `ge` | 大于等于 >= | `ge(User::getAge, 18)` |
| `lt` | 小于 < | `lt(User::getAge, 60)` |
| `le` | 小于等于 <= | `le(User::getAge, 60)` |
| `between` | BETWEEN | `between(User::getAge, 18, 60)` |
| `like` | LIKE '%值%' | `like(User::getName, "张")` |
| `likeLeft` | LIKE '%值' | `likeLeft(User::getName, "三")` |
| `likeRight` | LIKE '值%' | `likeRight(User::getName, "张")` |
| `in` | IN | `in(User::getId, 1, 2, 3)` |
| `notIn` | NOT IN | `notIn(User::getStatus, 0)` |
| `isNull` | IS NULL | `isNull(User::getEmail)` |
| `isNotNull` | IS NOT NULL | `isNotNull(User::getEmail)` |
| `orderByAsc` | 升序 | `orderByAsc(User::getAge)` |
| `orderByDesc` | 降序 | `orderByDesc(User::getCreateTime)` |

### 4.4 复杂条件

```java
@Test
public void testComplexQuery() {
    LambdaQueryWrapper<User> wrapper = new LambdaQueryWrapper<>();
    
    // WHERE (age > 20 OR gender = 1) AND username LIKE '%张%'
    wrapper.and(w -> w.gt(User::getAge, 20).or().eq(User::getGender, 1))
           .like(User::getUsername, "张");
    
    List<User> users = userMapper.selectList(wrapper);
    users.forEach(System.out::println);
}
```

### 4.5 动态条件

```java
@Test
public void testDynamicQuery(String keyword, Integer minAge, Integer maxAge) {
    LambdaQueryWrapper<User> wrapper = new LambdaQueryWrapper<>();
    
    // 关键字不为空时才添加条件
    wrapper.like(StringUtils.isNotBlank(keyword), User::getUsername, keyword);
    
    // 年龄范围
    wrapper.ge(minAge != null, User::getAge, minAge);
    wrapper.le(maxAge != null, User::getAge, maxAge);
    
    List<User> users = userMapper.selectList(wrapper);
    users.forEach(System.out::println);
}
```

## 5. 分页查询

### 5.1 配置分页插件

```java
package com.example.config;

import com.baomidou.mybatisplus.extension.plugins.MybatisPlusInterceptor;
import com.baomidou.mybatisplus.extension.plugins.inner.PaginationInnerInterceptor;
import org.springframework.context.annotation.Bean;
import org.springframework.context.annotation.Configuration;

@Configuration
public class MybatisPlusConfig {
    
    @Bean
    public MybatisPlusInterceptor mybatisPlusInterceptor() {
        MybatisPlusInterceptor interceptor = new MybatisPlusInterceptor();
        
        // 添加分页插件
        PaginationInnerInterceptor paginationInterceptor = new PaginationInnerInterceptor();
        paginationInterceptor.setDbType(DbType.MYSQL);
        paginationInterceptor.setOverflow(true);  // 溢出总页数后是否进行处理
        
        interceptor.addInnerInterceptor(paginationInterceptor);
        return interceptor;
    }
}
```

### 5.2 使用分页

```java
@Test
public void testPage() {
    // 第 1 页，每页 10 条
    Page<User> page = new Page<>(1, 10);
    
    // 执行分页查询
    Page<User> result = userMapper.selectPage(page, null);
    
    System.out.println("总记录数：" + result.getTotal());
    System.out.println("总页数：" + result.getPages());
    System.out.println("当前页：" + result.getCurrent());
    System.out.println("每页条数：" + result.getSize());
    System.out.println("数据列表：" + result.getRecords());
}
```

### 5.3 条件分页

```java

}
```

## 9. 与原生 MyBatis 对比

### 9.1 代码量对比

**原生 MyBatis**：
```java
// Mapper 接口
List<User> findByAge(@Param("age") Integer age);

// XML
<select id="findByAge" resultType="User">
    SELECT * FROM user WHERE age = #{age}
</select>

// 调用
List<User> users = mapper.findByAge(25);
```

**MyBatis-Plus**：
```java
// 无需编写 Mapper 方法和 XML
LambdaQueryWrapper<User> wrapper = new LambdaQueryWrapper<>();
wrapper.eq(User::getAge, 25);
List<User> users = mapper.selectList(wrapper);
```

### 9.2 选择建议

**使用 MyBatis-Plus 的场景**：
* 快速开发 CRUD 功能
* 项目规模中小型
* 团队熟悉 MyBatis-Plus

**使用原生 MyBatis 的场景**：
* 复杂 SQL 较多
* 对性能要求极高
* 已有大量 MyBatis 代码

**混合使用（推荐）**：
* 简单 CRUD 用 MP
* 复杂查询用手写 SQL
