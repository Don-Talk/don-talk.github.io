## 1. ActiveRecord 模式

### 1.1 什么是 ActiveRecord？

ActiveRecord 是一种设计模式，实体类继承 Model 后可以直接进行 CRUD 操作。

### 1.2 启用 ActiveRecord

```java
@Data
@TableName("user")
public class User extends Model<User> {
    
    @TableId(type = IdType.AUTO)
    private Long id;
    
    private String username;
    
    private String email;
    
    private Integer age;
}
```

### 1.3 使用示例

```java
@Test
public void testActiveRecord() {
    // 插入
    User user = new User();
    user.setUsername("AR用户");
    user.setEmail("ar@example.com");
    user.setAge(25);
    user.insert();  // 直接调用 insert
    
    System.out.println("生成的ID：" + user.getId());
    
    // 查询
    User queriedUser = user.selectById();
    System.out.println(queriedUser);
    
    // 更新
    user.setUsername("新名字");
    user.updateById();
    
    // 删除
    user.deleteById();
}
```

**优势**：代码更简洁，面向对象风格  
**缺点**：耦合度高，不推荐在复杂项目中使用

## 2. 链式调用

### 2.1 lambdaQuery - 查询链

```java
@Test
public void testLambdaQuery() {
    List<User> users = userService.lambdaQuery()
        .gt(User::getAge, 20)
        .eq(User::getGender, 1)
        .like(User::getUsername, "张")
        .orderByDesc(User::getCreateTime)
        .list();
    
    users.forEach(System.out::println);
}
```

### 2.2 lambdaUpdate - 更新链

```java
@Test
public void testLambdaUpdate() {
    boolean success = userService.lambdaUpdate()
        .set(User::getAge, 30)
        .set(User::getEmail, "new@example.com")
        .eq(User::getId, 1L)
        .update();
    
    System.out.println("更新结果：" + success);
}
```

### 2.3 分页链式调用

```java
@Test
public void testPageChain() {
    Page<User> page = userService.lambdaQuery()
        .gt(User::getAge, 20)
        .page(new Page<>(1, 10));
    
    System.out.println("总数：" + page.getTotal());
    page.getRecords().forEach(System.out::println);
}
```

### 2.4 单个对象查询

```java
@Test
public void testOneChain() {
    User user = userService.lambdaQuery()
        .eq(User::getUsername, "张三")
        .one();
    
    System.out.println(user);
}
```

### 2.5 统计数量

```java
@Test
public void testCountChain() {
    long count = userService.lambdaQuery()
        .gt(User::getAge, 20)
        .count();
    
    System.out.println("数量：" + count);
}
```

## 3. 分组查询与聚合

### 3.1 GROUP BY

```java
@Test
public void testGroupBy() {
    List<Map<String, Object>> result = userMapper.selectMaps(
        new QueryWrapper<User>()
            .select("gender, COUNT(*) as count, AVG(age) as avg_age")
            .groupBy("gender")
    );
    
    result.forEach(System.out::println);
}
```

**输出**：
```
{gender=1, count=50, avg_age=25.5}
{gender=2, count=30, avg_age=23.8}
```

**SQL**：
```sql
SELECT gender, COUNT(*) as count, AVG(age) as avg_age
FROM user
WHERE deleted=0
GROUP BY gender
```

### 3.2 HAVING

```java
@Test
public void testHaving() {
    List<Map<String, Object>> result = userMapper.selectMaps(
        new QueryWrapper<User>()
            .select("age, COUNT(*) as count")
            .groupBy("age")
            .having("COUNT(*) > {0}", 5)
    );
    
    result.forEach(System.out::println);
}
```

**SQL**：
```sql
SELECT age, COUNT(*) as count
FROM user
WHERE deleted=0
GROUP BY age
HAVING COUNT(*) > 5
```

### 3.3 聚合函数

```java
@Test
public void testAggregate() {
    // 最大年龄
    User maxAgeUser = userMapper.selectOne(
        new QueryWrapper<User>()
            .select("MAX(age) as age")
    );
    
    // 最小年龄
    User minAgeUser = userMapper.selectOne(
        new QueryWrapper<User>()
            .select("MIN(age) as age")
    );
    
    // 平均年龄
    User avgAgeUser = userMapper.selectOne(
        new QueryWrapper<User>()
            .select("AVG(age) as age")
    );
    
    // 总人数
    Integer count = userMapper.selectCount(null);
}
```

## 4. 子查询

### 4.1 in 子查询

```java
@Test
public void testInSubquery() {
    // 查询有订单的用户
    List<User> users = userMapper.selectList(
        new LambdaQueryWrapper<User>()
            .inSql(User::getId, 
                   "SELECT user_id FROM orders WHERE status = 1")
    );
}
```

**SQL**：
```sql
SELECT * FROM user 
WHERE id IN (SELECT user_id FROM orders WHERE status = 1)
AND deleted=0
```

### 4.2 exists 子查询

```java
@Test
public void testExistsSubquery() {
    List<User> users = userMapper.selectList(
        new LambdaQueryWrapper<User>()
            .exists("SELECT 1 FROM orders WHERE orders.user_id = user.id")
    );
}
```

### 4.3 比较子查询

```java
@Test
public void testCompareSubquery() {
    // 查询年龄大于平均年龄的用户
    List<User> users = userMapper.selectList(
        new LambdaQueryWrapper<User>()
            .gt(User::getAge, 
                new QueryWrapper<User>().select("AVG(age)").last("FROM user"))
    );
}
```

## 6. XML 自定义 SQL 混合使用

### 6.1 Mapper 接口

```java
public interface UserMapper extends BaseMapper<User> {
    
    /**
     * 复杂查询
     */
    List<UserVO> selectUserWithOrders(@Param("minAge") Integer minAge);
    
    /**
     * 复杂统计
     */
    Map<String, Object> selectUserStats();
}
```

### 6.2 XML 映射

```xml
<!-- UserMapper.xml -->

<!-- 关联查询 -->
<select id="selectUserWithOrders" resultType="com.example.vo.UserVO">
    SELECT 
        u.id,
        u.username,
        u.email,
        COUNT(o.id) as orderCount,
        SUM(o.amount) as totalAmount
    FROM user u
    LEFT JOIN orders o ON u.id = o.user_id
    <where>
        u.deleted = 0
        <if test="minAge != null">
            AND u.age &gt;= #{minAge}
        </if>
    </where>
    GROUP BY u.id, u.username, u.email
    HAVING COUNT(o.id) &gt; 0
</select>

<!-- 统计查询 -->
<select id="selectUserStats" resultType="map">
    SELECT 
        COUNT(*) as totalCount,
        AVG(age) as avgAge,
        MAX(age) as maxAge,
        MIN(age) as minAge
    FROM user
    WHERE deleted = 0
</select>
```

### 6.3 VO 类

```java
@Data
public class UserVO {
    private Long id;
    private String username;
    private String email;
    private Integer orderCount;
    private BigDecimal totalAmount;
}
```

### 6.4 调用

```java
@Test
public void testCustomSQL() {
    // 复杂查询
    List<UserVO> users = userMapper.selectUserWithOrders(20);
    users.forEach(System.out::println);
    
    // 统计查询
    Map<String, Object> stats = userMapper.selectUserStats();
    System.out.println(stats);
}
```

## 7. 动态表名

### 7.1 场景：分表

```java
public interface OrderMapper extends BaseMapper<Order> {
    
    /**
     * 查询指定月份的订单
     */
    @Select("SELECT * FROM order_${month} WHERE user_id = #{userId}")
    List<Order> selectByMonth(@Param("month") String month,
                              @Param("userId") Long userId);
}
```

**注意**：`${}` 不会转义，有 SQL 注入风险，需确保参数安全

### 7.2 动态表名插件

```java
@Bean
public MybatisPlusInterceptor mybatisPlusInterceptor() {
    MybatisPlusInterceptor interceptor = new MybatisPlusInterceptor();
    
    // 添加动态表名插件
    interceptor.addInnerInterceptor(new DynamicTableNameInnerInterceptor((dataSource, sql, tableName) -> {
        if ("order".equals(tableName)) {
            // 根据月份动态切换表名
            String month = DateUtil.format(new Date(), "yyyyMM");
            return "order_" + month;
        }
        return tableName;
    }));
    
    return interceptor;
}
```

## 9. 总结

### 9.1 核心知识点

✅ **ActiveRecord**：实体类直接 CRUD  
✅ **链式调用**：lambdaQuery、lambdaUpdate  
✅ **分组聚合**：GROUP BY、HAVING、聚合函数  
✅ **子查询**：inSql、exists、比较子查询  
✅ **原生 SQL**：@Select、@Update  
✅ **XML 混合**：复杂查询使用 XML  

### 9.2 最佳实践

📌 简单查询使用 Wrapper  
📌 复杂查询使用 XML  
📌 优先使用链式调用提高可读性  
📌 注意 SQL 注入风险  
📌 合理使用子查询优化性能
