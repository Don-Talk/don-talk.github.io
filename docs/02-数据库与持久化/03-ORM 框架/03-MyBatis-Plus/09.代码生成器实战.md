## 1. AutoGenerator 简介

### 1.1 什么是代码生成器？

AutoGenerator 是 MP 提供的代码生成工具，可以根据数据库表自动生成：
* Entity 实体类
* Mapper 接口
* Service 接口和实现类
* Controller 控制器
* Mapper XML 文件

**优势**：
* ✅ 提高开发效率
* ✅ 统一代码风格
* ✅ 减少重复劳动

## 2. 快速入门

### 2.1 添加依赖

```xml
<dependency>
    <groupId>com.baomidou</groupId>
    <artifactId>mybatis-plus-generator</artifactId>
    <version>3.5.3.1</version>
</dependency>

<!-- 模板引擎（三选一） -->
<dependency>
    <groupId>org.apache.velocity</groupId>
    <artifactId>velocity-engine-core</artifactId>
    <version>2.3</version>
</dependency>
```

**可选模板引擎**：
* Velocity（默认）
* Freemarker
* Beetl

### 2.2 基础配置

```java
package com.example.generator;

import com.baomidou.mybatisplus.generator.FastAutoGenerator;
import com.baomidou.mybatisplus.generator.config.OutputFile;

import java.util.Collections;

public class CodeGenerator {
    
    public static void main(String[] args) {
        FastAutoGenerator.create(
            "jdbc:mysql://localhost:3306/mp_demo?useSSL=false&serverTimezone=Asia/Shanghai",
            "root",
            "123456"
        )
        .globalConfig(builder -> {
            builder.author("Your Name")           // 作者
                   .outputDir("src/main/java");   // 输出目录
        })
        .packageConfig(builder -> {
            builder.parent("com.example")         // 父包名
                   .moduleName("system")          // 模块名
                   .entity("entity")
                   .mapper("mapper")
                   .service("service")
                   .controller("controller")
                   .pathInfo(Collections.singletonMap(
                       OutputFile.xml, "src/main/resources/mapper"));
        })
        .strategyConfig(builder -> {
            builder.addInclude("user", "order")   // 需要生成的表
                   .entityBuilder()
                   .enableLombok()                // 启用 Lombok
                   .enableTableFieldAnnotation()  // 启用字段注解
                   .controllerBuilder()
                   .enableRestStyle();            // 启用 REST 风格
        })
        .execute();
    }
}
```

运行后自动生成全套代码！

## 3. 详细配置

### 3.1 全局配置

```java
.globalConfig(builder -> {
    builder.author("Your Name")                    // 作者
           .outputDir("src/main/java")             // 输出目录
           .disableOpenDir()                       // 禁止打开输出目录
           .commentDate("yyyy-MM-dd")              // 注释日期格式
           .fileOverride();                        // 覆盖已生成文件
})
```

### 3.2 包配置

```java
.packageConfig(builder -> {
    builder.parent("com.example")                  // 父包名
           .moduleName("system")                   // 模块名
           .entity("model.entity")                 // Entity 包名
           .mapper("mapper")                       // Mapper 包名
           .service("service")                     // Service 包名
           .serviceImpl("service.impl")            // ServiceImpl 包名
           .controller("controller")               // Controller 包名
           .other("other")                         // 自定义文件包名
           .pathInfo(Collections.singletonMap(     // 路径配置
               OutputFile.xml, "src/main/resources/mapper"));
})
```

### 3.3 策略配置

```java
.strategyConfig(builder -> {
    builder.addInclude("user", "order", "product")  // 需要生成的表
           .addExclude("sys_log")                    // 排除的表
           
           // Entity 策略
           .entityBuilder()
           .enableLombok()                           // 启用 Lombok
           .enableTableFieldAnnotation()             // 启用字段注解
           .naming(NamingStrategy.underline_to_camel)  // 表名转驼峰
           .columnNaming(NamingStrategy.underline_to_camel)  // 字段名转驼峰
           .logicDeleteColumnName("deleted")         // 逻辑删除字段
           .addTableFills(                           // 自动填充字段
               new Column("create_time", FieldFill.INSERT),
               new Column("update_time", FieldFill.INSERT_UPDATE)
           )
           
           // Controller 策略
           .controllerBuilder()
           .enableRestStyle()                        // REST 风格
           .enableHyphenStyle();                     // 驼峰转连字符
})
```

## 4. 自定义模板

### 4.1 模板位置

默认模板位于：`templates/` 目录下

### 4.2 自定义 Entity 模板

创建 `templates/entity.java.vm`：

```velocity
package ${package.Entity};

import com.baomidou.mybatisplus.annotation.*;
import lombok.Data;
import java.io.Serializable;
import java.time.LocalDateTime;

/**
 * ${table.comment!}
 *
 * @author ${author}
 * @since ${date}
 */
@Data
@TableName("${table.name}")
#if(${table.convert})
public class ${entity} implements Serializable {
#else
public class ${entity} extends Model<${entity}> implements Serializable {
#end

    private static final long serialVersionUID = 1L;

#foreach($field in ${table.fields})
#if(${field.keyFlag})
    /**
     * ${field.comment!}
     */
    @TableId(value = "${field.annotationColumnName}", type = IdType.AUTO)
#elseif(${field.fill.toString()} != "DEFAULT")
    /**
     * ${field.comment!}
     */
    @TableField(value = "${field.annotationColumnName}", fill = FieldFill.${field.fill.toString()})
#else
    /**
     * ${field.comment!}
     */
    @TableField("${field.annotationColumnName}")
#end
    private ${field.propertyType} ${field.propertyName};

#end
}
```

### 4.3 使用自定义模板

```java
.templateConfig(builder -> {
    builder.entity("/templates/entity.java")       // 自定义 Entity 模板
           .mapper("/templates/mapper.java")       // 自定义 Mapper 模板
           .service("/templates/service.java")     // 自定义 Service 模板
           .serviceImpl("/templates/serviceImpl.java")
           .controller("/templates/controller.java")
           .xml("/templates/mapper.xml");
})
```

## 5. 批量生成

### 5.1 生成所有表

```java
.strategyConfig(builder -> {
    // 不指定表，生成所有表
    builder.addInclude(".*")  // 正则表达式匹配所有表
})
```

### 5.2 按模块生成

```java
// 生成用户模块
FastAutoGenerator.create(...)
    .strategyConfig(builder -> {
        builder.addInclude("sys_user", "sys_role", "sys_permission");
    })
    .execute();

// 生成订单模块
FastAutoGenerator.create(...)
    .strategyConfig(builder -> {
        builder.addInclude("order", "order_item", "order_payment");
    })
    .execute();
```

## 7. 实战：完整示例

### 7.1 生成器类

```java
package com.example.generator;

import com.baomidou.mybatisplus.generator.FastAutoGenerator;
import com.baomidou.mybatisplus.generator.config.OutputFile;
import com.baomidou.mybatisplus.generator.engine.FreemarkerTemplateEngine;

import java.util.Collections;

public class MpGenerator {
    
    public static void main(String[] args) {
        String url = "jdbc:mysql://localhost:3306/mp_demo?useSSL=false&serverTimezone=Asia/Shanghai";
        String username = "root";
        String password = "123456";
        
        FastAutoGenerator.create(url, username, password)
        
        // 全局配置
        .globalConfig(builder -> {
            builder.author("DonTalk")
                   .outputDir(System.getProperty("user.dir") + "/src/main/java")
                   .commentDate("yyyy-MM-dd")
                   .disableOpenDir();
        })
        
        // 包配置
        .packageConfig(builder -> {
            builder.parent("com.example")
                   .moduleName("business")
                   .entity("model.entity")
                   .mapper("mapper")
                   .service("service")
                   .serviceImpl("service.impl")
                   .controller("controller")
                   .pathInfo(Collections.singletonMap(
                       OutputFile.xml, 
                       System.getProperty("user.dir") + "/src/main/resources/mapper"));
        })
        
        // 策略配置
        .strategyConfig(builder -> {
            builder.addInclude("product", "category", "brand")
                   .addExclude("sys_.*")  // 排除系统表
                   
                   .entityBuilder()
                   .enableLombok()
                   .enableTableFieldAnnotation()
                   .logicDeleteColumnName("deleted")
                   .addTableFills(
                       new com.baomidou.mybatisplus.generator.config.po.TableFill(
                           "create_time", 
                           com.baomidou.mybatisplus.annotation.FieldFill.INSERT),
                       new com.baomidou.mybatisplus.generator.config.po.TableFill(
                           "update_time", 
                           com.baomidou.mybatisplus.annotation.FieldFill.INSERT_UPDATE)
                   )
                   
                   .controllerBuilder()
                   .enableRestStyle()
                   .enableHyphenStyle();
        })
        
        // 使用 Freemarker 引擎
        .templateEngine(new FreemarkerTemplateEngine())
        
        .execute();
        
        System.out.println("代码生成完成！");
    }
}
```

## 8. 常见问题

### 8.1 中文注释乱码

**解决**：设置编码

```java
.globalConfig(builder -> {
    builder.charset("UTF-8");
})
```

### 8.2 表名带前缀

**问题**：表名 `sys_user`，生成的类名 `SysUser`，想去掉 `Sys` 前缀

**解决**：

```java
.strategyConfig(builder -> {
    builder.entityBuilder()
           .addTablePrefix("sys_");  // 去掉表前缀
})
```

### 8.3 字段类型映射错误

**问题**：数据库 `datetime` 映射为 `Date`，想要 `LocalDateTime`

**解决**：

```java
.strategyConfig(builder -> {
    builder.entityBuilder()
           .convertTypes()
           .addConvert(new TypeConverts() {
               @Override
               public ITypeConvert<?> getTypeConvert(ProcessConfig config, TableField field) {
                   if ("datetime".equalsIgnoreCase(field.getType())) {
                       return DbColumnType.LOCAL_DATE_TIME;
                   }
                   return null;
               }
           });
})
```
