# 01Linux环境安装Oracle 19c

**学习目标：**
> * 掌握 Oracle 19c 在 Linux 上的安装准备
> * 能够完成图形化和静默安装
> * 学会配置环境变量和启动服务
> * 了解常见安装问题的解决方法

---

## 1. 系统要求

### 1.1 硬件要求

| 组件 | 最低要求 | 推荐配置 |
|------|---------|---------|
| **内存** | 2 GB | 8 GB+ |
| **Swap** | 1.5×RAM（≤2GB） | 与 RAM 相同（>8GB） |
| **磁盘空间** | 10 GB | 50 GB+ |
| **/tmp** | 1 GB | 2 GB |

```bash
# 检查内存
grep MemTotal /proc/meminfo

# 检查 Swap
grep SwapTotal /proc/meminfo

# 检查磁盘空间
df -h /
df -h /tmp
```

### 1.2 软件要求

**支持的 Linux 版本**：
- Oracle Linux 7.x, 8.x
- Red Hat Enterprise Linux 7.x, 8.x
- CentOS 7.x, 8.x
- SUSE Linux Enterprise Server 12, 15

```bash
# 检查操作系统版本
cat /etc/os-release

# 检查内核版本
uname -r

# 检查必要的包
rpm -q binutils compat-libcap1 gcc gcc-c++ glibc glibc-devel \
  ksh libaio libaio-devel libgcc libstdc++ libstdc++-devel \
  libXi libXtst make sysstat unixODBC unixODBC-devel
```

## 2. 安装前准备

### 2.1 创建用户和组

```bash
# 创建 oinstall 和 dba 组
groupadd oinstall
groupadd dba
groupadd oper

# 创建 oracle 用户
useradd -g oinstall -G dba,oper oracle

# 设置密码
passwd oracle
```

### 2.2 配置内核参数

编辑 `/etc/sysctl.conf`：

```bash
# Oracle 内核参数
fs.aio-max-nr = 1048576
fs.file-max = 6815744
kernel.shmall = 2097152
kernel.shmmax = 4294967295
kernel.shmmni = 4096
kernel.sem = 250 32000 100 128
net.ipv4.ip_local_port_range = 9000 65500
net.core.rmem_default = 262144
net.core.rmem_max = 4194304
net.core.wmem_default = 262144
net.core.wmem_max = 1048576
```

**应用配置**：
```bash
sysctl -p
```

**参数说明**：
- `kernel.shmmax`：共享内存段最大值（建议设置为物理内存的 50%）
- `kernel.shmall`：共享内存页总数
- `fs.file-max`：系统最大文件句柄数

### 2.3 配置 Shell 限制

编辑 `/etc/security/limits.conf`：

```bash
# Oracle 用户限制
oracle   soft   nofile    1024
oracle   hard   nofile    65536
oracle   soft   nproc     2047
oracle   hard   nproc     16384
oracle   soft   stack     10240
oracle   hard   stack     32768
```

编辑 `/etc/pam.d/login`，确保包含：
```bash
session    required     pam_limits.so
```

### 2.4 创建目录结构

```bash
# 创建 Oracle Base 和 Inventory 目录
mkdir -p /u01/app/oracle
mkdir -p /u01/app/oraInventory

# 设置权限
chown -R oracle:oinstall /u01/app/oracle
chown -R oracle:oinstall /u01/app/oraInventory
chmod -R 775 /u01/app/oracle
chmod -R 775 /u01/app/oraInventory
```

### 2.5 配置环境变量

切换到 oracle 用户，编辑 `~/.bash_profile`：

```bash
# Oracle 环境变量
export ORACLE_BASE=/u01/app/oracle
export ORACLE_HOME=$ORACLE_BASE/product/19.0.0/dbhome_1
export ORACLE_SID=ORCL
export PATH=$ORACLE_HOME/bin:$PATH
export LD_LIBRARY_PATH=$ORACLE_HOME/lib:/lib:/usr/lib
export CLASSPATH=$ORACLE_HOME/jlib:$ORACLE_HOME/rdbms/jlib

# 别名
alias sqlplus='rlwrap sqlplus'
alias rman='rlwrap rman'
```

**生效配置**：
```bash
source ~/.bash_profile
```

### 2.6 禁用防火墙和 SELinux（可选）

```bash
# 临时禁用防火墙
systemctl stop firewalld
systemctl disable firewalld

# 禁用 SELinux
setenforce 0
sed -i 's/SELINUX=enforcing/SELINUX=disabled/' /etc/selinux/config
```

**生产环境建议**：配置防火墙规则而非完全禁用

```bash
# 开放 Oracle 端口
firewall-cmd --permanent --add-port=1521/tcp
firewall-cmd --permanent --add-port=5500/tcp
firewall-cmd --reload
```

## 3. 下载 Oracle 19c

### 3.1 下载地址

从 Oracle 官网下载：
- https://www.oracle.com/database/technologies/oracle19c-linux-downloads.html

需要 Oracle 账号（免费注册）

### 3.2 上传到服务器

```bash
# 使用 scp 上传
scp LINUX.X64_193000_db_home.zip oracle@server:/u01/app/oracle/

# 或使用 FTP/SFTP 工具
```

### 3.3 解压安装包

```bash
# 切换到 oracle 用户
su - oracle

# 创建 Oracle Home 目录
mkdir -p $ORACLE_HOME

# 解压到 Oracle Home
cd $ORACLE_HOME
unzip -q /u01/app/oracle/LINUX.X64_193000_db_home.zip
```

**注意**：必须解压到 `$ORACLE_HOME` 目录

## 4. 图形化安装（OUI）

## 5. 静默安装（推荐）

静默安装适合自动化部署和无图形界面环境。

### 5.1 创建响应文件

创建 `/u01/app/oracle/db_install.rsp`：

```ini
####################################################################
## Copyright(c) Oracle Corporation 1998,2019. All rights reserved.##
####################################################################

#-------------------------------------------------------------------------------
# 安装选项
#-------------------------------------------------------------------------------
oracle.install.option=INSTALL_DB_SWONLY
UNIX_GROUP_NAME=oinstall
INVENTORY_LOCATION=/u01/app/oraInventory
ORACLE_HOME=/u01/app/oracle/product/19.0.0/dbhome_1
ORACLE_BASE=/u01/app/oracle
oracle.install.db.InstallEdition=EE
oracle.install.db.OSDBA_GROUP=dba
oracle.install.db.OSOPER_GROUP=oper
oracle.install.db.OSBACKUPDBA_GROUP=dba
oracle.install.db.OSDGDBA_GROUP=dba
oracle.install.db.OSKMDBA_GROUP=dba
oracle.install.db.OSRACDBA_GROUP=dba

#-------------------------------------------------------------------------------
# 安全选项
#-------------------------------------------------------------------------------
DECLINE_SECURITY_UPDATES=true
SECURITY_UPDATES_VIA_MYORACLESUPPORT=false

#-------------------------------------------------------------------------------
# 自动内存管理
#-------------------------------------------------------------------------------
oracle.install.db.config.starterdb.memoryOption=true
oracle.install.db.config.starterdb.memoryLimit=4096

#-------------------------------------------------------------------------------
# 数据库配置
#-------------------------------------------------------------------------------
oracle.install.db.config.starterdb.type=GENERAL_PURPOSE
oracle.install.db.config.starterdb.globalDBName=ORCL
oracle.install.db.config.starterdb.SID=ORCL
oracle.install.db.config.starterdb.characterSet=AL32UTF8
oracle.install.db.config.starterdb.storageType=FILE_SYSTEM_STORAGE
oracle.install.db.config.starterdb.fileSystemStorage.dataLocation=/u01/app/oracle/oradata
oracle.install.db.config.starterdb.fileSystemStorage.recoveryLocation=/u01/app/oracle/fast_recovery_area
oracle.install.db.config.starterdb.enableArchive=true
oracle.install.db.config.starterdb.autoMemOption=true
oracle.install.db.config.starterdb.password.ALL=Oracle123
```

### 5.2 执行静默安装

```bash
# 切换到 oracle 用户
su - oracle

# 执行安装
cd $ORACLE_HOME
./runInstaller -silent \
  -responseFile /u01/app/oracle/db_install.rsp \
  -ignorePrereq \
  -waitforcompletion
```

### 5.3 执行 root 脚本

```bash
# 以 root 用户执行
/u01/app/oraInventory/orainstRoot.sh
/u01/app/oracle/product/19.0.0/dbhome_1/root.sh
```

### 5.4 创建数据库（可选）

如果选择了只安装软件，需要手动创建数据库：

```bash
# 使用 DBCA 创建数据库
dbca -silent \
  -createDatabase \
  -templateName General_Purpose.dbc \
  -gdbname ORCL \
  -sid ORCL \
  -responseFile NO_VALUE \
  -characterSet AL32UTF8 \
  -memoryPercentage 40 \
  -databaseType OLTP \
  -emConfiguration NONE \
  -datafileDestination /u01/app/oracle/oradata \
  -recoveryAreaDestination /u01/app/oracle/fast_recovery_area \
  -storageType FS \
  -automaticMemoryManagement true \
  -totalMemory 4096 \
  -sampleSchema false \
  -enableArchive true
```

## 6. 验证安装

### 6.1 检查进程

```bash
# 查看 Oracle 进程
ps -ef | grep pmon

# 应该看到类似输出：
# oracle  12345     1  0 10:00 ?        00:00:00 ora_pmon_ORCL
```

### 6.2 连接数据库

```bash
# 使用 SQL*Plus 连接
sqlplus / as sysdba

# 查询数据库信息
SELECT instance_name, status, version FROM v$instance;
SELECT name, open_mode FROM v$database;
```

**预期输出**：
```
INSTANCE_NAME    STATUS       VERSION
---------------- ------------ -----------------
ORCL             OPEN         19.0.0.0.0

NAME      OPEN_MODE
--------- --------------------
ORCL      READ WRITE
```

### 6.3 检查监听器

```bash
# 启动监听器
lsnrctl start

# 查看状态
lsnrctl status

# 应该看到服务 ORCL 已注册
```

```bash
# 使用 TNS 连接
sqlplus sys/Oracle123@ORCL as sysdba

# 或使用 EZCONNECT
sqlplus sys/Oracle123@localhost:1521/ORCL as sysdba
```

## 7. 配置开机自启动

### 7.1 创建 systemd 服务

创建 `/etc/systemd/system/oracle-db.service`：

```ini
[Unit]
Description=Oracle Database Service
After=network.target

[Service]
Type=forking
User=oracle
Group=oinstall
ExecStart=/u01/app/oracle/product/19.0.0/dbhome_1/bin/dbstart /u01/app/oracle
ExecStop=/u01/app/oracle/product/19.0.0/dbhome_1/bin/dbshut /u01/app/oracle
TimeoutSec=300

[Install]
WantedBy=multi-user.target
```

**启用服务**：
```bash
systemctl daemon-reload
systemctl enable oracle-db
systemctl start oracle-db
```

### 7.2 配置 oratab

编辑 `/etc/oratab`：

```bash
ORCL:/u01/app/oracle/product/19.0.0/dbhome_1:Y
```

**最后一个字段**：
- `Y`：开机自动启动
- `N`：不自动启动

## 8. 常见问题

### 8.1 交换空间不足

**错误**：
```
PRVF-7573 : Insufficient swap space
```

**解决**：
```bash
# 创建 swap 文件
dd if=/dev/zero of=/swapfile bs=1M count=4096
mkswap /swapfile
swapon /swapfile

# 永久生效，添加到 /etc/fstab
echo '/swapfile swap swap defaults 0 0' >> /etc/fstab
```

### 8.2 缺少依赖包

**错误**：
```
Package xxx is missing
```

**解决**：
```bash
# 安装缺失的包
yum install -y binutils compat-libcap1 gcc gcc-c++ glibc glibc-devel \
  ksh libaio libaio-devel libgcc libstdc++ libstdc++-devel \
  libXi libXtst make sysstat unixODBC unixODBC-devel
```

### 8.3 权限问题

**错误**：
```
Permission denied
```

**解决**：
```bash
# 修正权限
chown -R oracle:oinstall /u01/app/oracle
chmod -R 775 /u01/app/oracle
```

### 8.4 DISPLAY 未设置

**错误**：
```
DISPLAY not set
```

**解决**：
```bash
# 配置 X11 转发
export DISPLAY=localhost:0.0

# 或使用静默安装
./runInstaller -silent -responseFile /path/to/response.rsp
```

### 8.5 监听器无法启动

**错误**：
```
TNS-12541: TNS:no listener
```

**解决**：
```bash
# 检查 listener.ora
cat $ORACLE_HOME/network/admin/listener.ora

# 重新启动监听器
lsnrctl stop
lsnrctl start

# 查看日志
tail -f $ORACLE_BASE/diag/tnslsnr/hostname/listener/alert/log.xml
```

## 9. 卸载 Oracle

### 9.1 使用 OUI 卸载

```bash
# 启动卸载程序
$ORACLE_HOME/deinstall/deinstall

# 按照提示操作
```

### 9.2 手动卸载

```bash
# 1. 停止数据库和监听器
sqlplus / as sysdba <<EOF
SHUTDOWN IMMEDIATE;
EXIT;
EOF

lsnrctl stop

# 2. 删除 Oracle Home
rm -rf /u01/app/oracle/product/19.0.0/dbhome_1

# 3. 删除 Oracle Base（可选）
rm -rf /u01/app/oracle

# 4. 删除 Inventory
rm -rf /u01/app/oraInventory

# 5. 清理 /etc 下的配置
rm -f /etc/oraInst.loc
rm -f /etc/oratab
rm -rf /etc/oracle

# 6. 删除用户和组（可选）
userdel -r oracle
groupdel oinstall
groupdel dba
```
