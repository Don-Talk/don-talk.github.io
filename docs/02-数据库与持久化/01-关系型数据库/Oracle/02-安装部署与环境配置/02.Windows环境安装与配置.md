> * 掌握 Oracle 19c 在 Windows 上的安装流程

## 2. 安装前准备

### 2.1 下载 Oracle 19c

从 Oracle 官网下载 Windows 版本：
- https://www.oracle.com/database/technologies/oracle19c-windows-downloads.html

下载文件：`WINDOWS.X64_193000_db_home.zip`

### 2.2 创建目录结构

```powershell
# 以管理员身份运行 PowerShell
mkdir C:\app\oracle
mkdir C:\app\oracle\product\19.0.0\dbhome_1
```

### 2.3 解压安装包

```powershell
# 解压到 Oracle Home 目录
Expand-Archive -Path "C:\Downloads\WINDOWS.X64_193000_db_home.zip" `
               -DestinationPath "C:\app\oracle\product\19.0.0\dbhome_1"
```

- 等待检查完成

- 等待安装完成（约 30-60 分钟）

#### 步骤 7：完成
- 记录 Enterprise Manager URL

## 4. 静默安装

### 4.1 创建响应文件

创建 `C:\app\db_install.rsp`：

```ini
oracle.install.option=INSTALL_DB_SWONLY
ORACLE_BASE=C:\app\oracle
ORACLE_HOME=C:\app\oracle\product\19.0.0\dbhome_1
oracle.install.db.InstallEdition=EE
oracle.install.db.OSDBA_GROUP=ORA_DBA
oracle.install.db.OSOPER_GROUP=ORA_OPER
DECLINE_SECURITY_UPDATES=true
```

### 4.2 执行静默安装

```powershell
# 以管理员身份运行
cd C:\app\oracle\product\19.0.0\dbhome_1

.\setup.exe -silent `
  -responseFile "C:\app\db_install.rsp" `
  -waitforcompletion
```

### 4.3 创建数据库

```powershell
# 使用 DBCA 创建数据库
dbca -silent `
  -createDatabase `
  -templateName General_Purpose.dbc `
  -gdbname ORCL `
  -sid ORCL `
  -characterSet AL32UTF8 `
  -memoryPercentage 40 `
  -databaseType OLTP `
  -emConfiguration NONE
```

## 5. 配置环境变量

### 5.1 设置系统环境变量

```powershell
# 以管理员身份运行 PowerShell

# 设置 ORACLE_HOME
[Environment]::SetEnvironmentVariable("ORACLE_HOME", "C:\app\oracle\product\19.0.0\dbhome_1", "Machine")

# 设置 PATH
$oldPath = [Environment]::GetEnvironmentVariable("Path", "Machine")
$newPath = "$oldPath;C:\app\oracle\product\19.0.0\dbhome_1\bin"
[Environment]::SetEnvironmentVariable("Path", $newPath, "Machine")

# 设置 TNS_ADMIN（可选）
[Environment]::SetEnvironmentVariable("TNS_ADMIN", "C:\app\oracle\product\19.0.0\dbhome_1\network\admin", "Machine")
```

### 5.2 验证配置

```powershell
# 重新打开 PowerShell
echo $env:ORACLE_HOME
echo $env:Path

# 测试 SQL*Plus
sqlplus /nolog
```

## 6. 服务管理

### 6.1 Oracle 服务列表

Oracle 在 Windows 上注册为多个服务：

| 服务名 | 作用 |
|--------|------|
| `OracleService<SID>` | 数据库实例服务 |
| `OracleOraDB19Home1TNSListener` | 监听器服务 |
| `OracleVssWriter<SID>` | VSS 写入器（备份用） |
| `OracleJobScheduler<SID>` | 作业调度器 |

### 6.2 启动和停止服务

```powershell
# 启动数据库服务
Start-Service OracleServiceORCL

# 停止数据库服务
Stop-Service OracleServiceORCL

# 启动监听器
Start-Service OracleOraDB19Home1TNSListener

# 停止监听器
Stop-Service OracleOraDB19Home1TNSListener

# 查看所有 Oracle 服务
Get-Service | Where-Object {$_.DisplayName -like "Oracle*"}
```

### 6.3 设置自动启动

```powershell
# 设置服务为自动启动
Set-Service OracleServiceORCL -StartupType Automatic
Set-Service OracleOraDB19Home1TNSListener -StartupType Automatic

# 设置为手动启动
Set-Service OracleServiceORCL -StartupType Manual
```

### 6.4 使用命令行工具

```powershell
# 使用 oradim 管理实例
oradim -startup -sid ORCL
oradim -shutdown -sid ORCL

# 使用 lsnrctl 管理监听器
lsnrctl start
lsnrctl stop
lsnrctl status
```

## 7. 连接测试

### 7.1 本地连接

```powershell
# 使用 SQL*Plus
sqlplus / as sysdba

# 查询数据库信息
SELECT instance_name, status FROM v$instance;
SELECT name, open_mode FROM v$database;
```

### 7.2 远程连接配置

编辑 `C:\app\oracle\product\19.0.0\dbhome_1\network\admin\listener.ora`：

```ini
LISTENER =
  (DESCRIPTION_LIST =
    (DESCRIPTION =
      (ADDRESS = (PROTOCOL = TCP)(HOST = localhost)(PORT = 1521))
    )
  )
```

编辑 `tnsnames.ora`：

```ini
ORCL =
  (DESCRIPTION =
    (ADDRESS = (PROTOCOL = TCP)(HOST = localhost)(PORT = 1521))
    (CONNECT_DATA =
      (SERVER = DEDICATED)
      (SERVICE_NAME = ORCL)
    )
  )
```

### 7.3 测试连接

```powershell
# 使用 TNS 名称连接
sqlplus sys/password@ORCL as sysdba

# 使用 EZCONNECT
sqlplus sys/password@localhost:1521/ORCL as sysdba
```

## 8. 防火墙配置

### 8.1 开放端口

```powershell
# 以管理员身份运行

# 开放监听器端口
New-NetFirewallRule -DisplayName "Oracle Listener" `
  -Direction Inbound `
  -Protocol TCP `
  -LocalPort 1521 `
  -Action Allow

# 开放 EM 端口
New-NetFirewallRule -DisplayName "Oracle EM" `
  -Direction Inbound `
  -Protocol TCP `
  -LocalPort 5500 `
  -Action Allow
```

### 8.2 验证端口

```powershell
# 检查端口是否监听
netstat -an | findstr "1521"
```

## 10. 常见问题

### 10.1 服务无法启动

**症状**：
- OracleServiceORCL 启动失败
- 错误代码 1067

**解决**：
```powershell
# 查看事件日志
Get-EventLog -LogName Application -Source "Oracle*" -Newest 10

# 检查 alert.log
Get-Content "C:\app\oracle\diag\rdbms\orcl\ORCL\trace\alert_ORCL.log" -Tail 50

# 常见原因：
# 1. 内存不足
# 2. 参数文件损坏
# 3. 控制文件丢失
```

### 10.2 监听器无法启动

**症状**：
- TNS-12541: TNS:no listener

**解决**：
```powershell
# 检查 listener.ora
Get-Content "C:\app\oracle\product\19.0.0\dbhome_1\network\admin\listener.ora"

# 重新启动监听器
lsnrctl stop
lsnrctl start

# 查看监听器日志
Get-Content "C:\app\oracle\diag\tnslsnr\hostname\listener\alert\log.xml" -Tail 50
```

### 10.3 ORA-12560 协议适配器错误

**原因**：
- ORACLE_SID 未设置
- 服务未启动

**解决**：
```powershell
# 设置 ORACLE_SID
$env:ORACLE_SID = "ORCL"

# 启动服务
Start-Service OracleServiceORCL
```

### 10.4 权限问题

**症状**：
- 访问被拒绝
- 无法创建文件

**解决**：
```powershell
# 以管理员身份运行 PowerShell
# 确保 oracle 用户在 ORA_DBA 组中
Add-LocalGroupMember -Group "ORA_DBA" -Member "username"
```

## 11. 卸载 Oracle

### 11.1 使用 Universal Installer

```powershell
# 启动卸载程序
cd C:\app\oracle\product\19.0.0\dbhome_1\deinstall
.\deinstall.bat
```

### 11.2 手动卸载

```powershell
# 1. 停止所有 Oracle 服务
Get-Service | Where-Object {$_.DisplayName -like "Oracle*"} | Stop-Service

# 2. 删除服务
sc delete OracleServiceORCL
sc delete OracleOraDB19Home1TNSListener

# 3. 删除目录
Remove-Item -Recurse -Force "C:\app\oracle"

# 4. 删除环境变量
[Environment]::SetEnvironmentVariable("ORACLE_HOME", $null, "Machine")
[Environment]::SetEnvironmentVariable("ORACLE_SID", $null, "Machine")

# 5. 清理注册表（谨慎操作）
# 使用 regedit 手动删除 Oracle 相关键值
```

## 12. 最佳实践

### 12.1 安装建议

✅ **目录规划**：
- Oracle Base: `C:\app\oracle`
- Oracle Home: `C:\app\oracle\product\19.0.0\dbhome_1`
