# Oracle 认证考试指南

## 📋 认证路径

### Oracle Database 认证路径

```
OCA (Associate)
    ↓
OCP (Professional)
    ↓
OCM (Master)
```

---

## 🎯 OCA - Oracle Certified Associate

### 考试信息
- **考试代码**：1Z0-071
- **考试科目**：Oracle Database SQL
- **题目数量**：约 85 题
- **考试时间**：120 分钟
- **通过分数**：约 60%

### 考试内容
1. **SQL 基础**
   - SELECT 语句
   - 过滤和排序
   - 单行函数
   - 多表连接

2. **高级 SQL**
   - 分组函数
   - 子查询
   - 集合操作
   - DML 语句

3. **数据定义**
   - 创建和管理表
   - 约束
   - 视图
   - 序列、索引、同义词

### 备考建议
✅ 完成模块 03（SQL基础与高级查询）  
✅ 大量练习 SQL 题目  
✅ 熟悉常用函数和语法  
✅ 模拟考试练习  

---

## 🎯 OCP - Oracle Certified Professional

### 考试信息
- **考试代码**：1Z0-082 + 1Z0-083
- **考试科目**：
  - 1Z0-082: Administration Workshop I
  - 1Z0-083: Administration Workshop II
- **题目数量**：各约 85 题
- **考试时间**：各 120 分钟

### 考试内容

#### 1Z0-082 (Administration I)
1. **架构组件**
   - 内存结构
   - 后台进程
   - 存储结构

2. **数据库管理**
   - 创建数据库
   - 表空间管理
   - 用户和权限

3. **网络配置**
   - 监听器配置
   - 网络连接

#### 1Z0-083 (Administration II)
1. **备份恢复**
   - RMAN 备份
   - 恢复策略
   - 闪回技术

2. **性能调优**
   - SQL 调优
   - 等待事件
   - AWR/ASH

3. **高可用**
   - Data Guard
   - RAC 基础

### 备考建议
✅ 完成所有核心模块（01-10）  
✅ 实际动手操作练习  
✅ 深入理解架构原理  
✅ 掌握故障诊断技能  

---

## 🎯 OCM - Oracle Certified Master

### 考试信息
- **考试形式**：实操考试（2天）
- **考试内容**：综合场景测试
- **前置要求**：必须先获得 OCP

### 考试内容
1. **服务器配置**
2. **数据库创建和配置**
3. **网络配置**
4. **数据库安全**
5. **网络服务优化**
6. **SQL 调优**
7. **Data Guard**
8. **资源管理**
9. **数据库维护**
10. **备份恢复**

### 备考建议
✅ 丰富的实际工作经验  
✅ 熟练掌握所有 DBA 任务  
✅ 参加官方培训课程  
✅ 模拟实战练习

## 💡 备考技巧

### 通用建议
1. **制定学习计划** - 至少 2-3 个月准备时间
2. **理论学习 + 实践操作** - 两者缺一不可
3. **做模拟题** - 熟悉题型和节奏
4. **加入学习社区** - 交流经验和疑问
5. **定期复习** - 巩固知识点

### 考试技巧
- ⏰ **时间管理** - 合理分配每题时间
- 🎯 **排除法** - 先排除明显错误选项
- ✅ **标记难题** - 先做有把握的题目
- 📝 **仔细阅读** - 注意题目细节
