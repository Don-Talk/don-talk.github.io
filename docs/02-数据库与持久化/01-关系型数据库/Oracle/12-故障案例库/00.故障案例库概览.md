## 案例分类

### 🔴 紧急故障（P0）
- 数据库无法启动
- 数据文件损坏
- 严重性能退化

### 🟡 重要故障（P1）
- 表空间满
- 死锁频繁发生
- 备份失败

### 🟢 一般问题（P2）
- SQL 性能下降
- 连接数过多
- 告警日志错误
