# PLsql基础

## 一、PL/SQL 基础认知

**什么是 PL/SQL**

- 过程化 + SQL 结合的语言
- 只能在 Oracle 里运行

**PL/SQL 程序结构**

- 块结构：`DECLARE` → `BEGIN` → `EXCEPTION` → `END`

**运行环境**

- SQL Developer、PL/SQL Developer、DBeaver

**优点**

- 执行快、减少网络交互、可复用、安全
