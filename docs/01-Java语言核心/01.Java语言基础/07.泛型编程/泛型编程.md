# 泛型编程

## 一、泛型概述

### 1.1 什么是泛型？

泛型（Generics）是 JDK 5 引入的特性，允许在定义类、接口和方法时使用类型参数。

**优点：**
- 类型安全：编译期检查类型
- 消除强制转换
- 代码复用

### 1.2 没有泛型的问题

```java
// 没有泛型
List list = new ArrayList();
list.add("Hello");
list.add(123);

String str = (String) list.get(0);  // 需要强制转换
Integer num = (Integer) list.get(1);  // 可能ClassCastException
```

## 二、泛型类

### 2.1 定义泛型类

```java
public class Box<T> {
    private T content;
    
    public void set(T content) {
        this.content = content;
    }
    
    public T get() {
        return content;
    }
}

// 使用
Box<String> stringBox = new Box<>();
stringBox.set("Hello");
String str = stringBox.get();  // 不需要强制转换

Box<Integer> intBox = new Box<>();
intBox.set(123);
Integer num = intBox.get();
```

### 2.2 多个类型参数

```java
public class Pair<K, V> {
    private K key;
    private V value;
    
    public Pair(K key, V value) {
        this.key = key;
        this.value = value;
    }
    
    public K getKey() { return key; }
    public V getValue() { return value; }
}

// 使用
Pair<String, Integer> pair = new Pair<>("Age", 20);
```

## 三、泛型方法

### 3.1 定义泛型方法

```java
public class Utils {
    
    // 泛型方法
    public static <T> T getLast(T[] array) {
        if (array == null || array.length == 0) {
            return null;
        }
        return array[array.length - 1];
    }
    
    // 多个类型参数
    public static <K, V> void printEntry(K key, V value) {
        System.out.println(key + " = " + value);
    }
}

// 使用
String last = Utils.getLast(new String[]{"A", "B", "C"});  // "C"
Utils.printEntry("Name", "张三");
```

### 3.2 泛型方法与泛型类的区别

```java
public class GenericClass<T> {
    
    // 这是泛型类的方法，T是类的类型参数
    public void method1(T item) {
    }
    
    // 这是泛型方法，U是方法的类型参数
    public <U> void method2(U item) {
    }
}
```

## 四、类型通配符

### 4.1 无界通配符 `?`

```java
public void printList(List<?> list) {
    for (Object item : list) {
        System.out.println(item);
    }
}

// 可以接受任何类型的List
printList(new ArrayList<String>());
printList(new ArrayList<Integer>());
```

### 4.2 上界通配符 `? extends T`

```java
// 只读，不能写入（除了null）
public double sum(List<? extends Number> numbers) {
    double sum = 0;
    for (Number num : numbers) {
        sum += num.doubleValue();
    }
    return sum;
}

// 使用
sum(Arrays.asList(1, 2, 3));           // Integer
sum(Arrays.asList(1.1, 2.2, 3.3));     // Double
```

### 4.3 下界通配符 `? super T`

```java
// 可以写入T及其子类
public void addNumbers(List<? super Integer> list) {
    list.add(1);
    list.add(2);
    // list.add(1.5);  // 错误！不能添加Double
}

// 使用
addNumbers(new ArrayList<Integer>());
addNumbers(new ArrayList<Number>());
addNumbers(new ArrayList<Object>());
```

### 4.4 PECS 原则

- **Producer Extends**：如果只需要读取，使用 `? extends T`
- **Consumer Super**：如果需要写入，使用 `? super T`

## 五、类型擦除

### 5.1 什么是类型擦除？

泛型信息只在编译期存在，运行时会擦除为原始类型。

```java
List<String> list1 = new ArrayList<>();
List<Integer> list2 = new ArrayList<>();

// 运行时类型相同
System.out.println(list1.getClass() == list2.getClass());  // true
```

### 5.2 类型擦除的影响

```java
// 错误！不能创建泛型数组
// T[] array = new T[10];

// 错误！不能使用 instanceof
// if (obj instanceof List<String>)

// 错误！不能捕获泛型异常
// catch (T e)

// 正确做法
public class GenericClass<T> {
    private Object[] array;
    
    @SuppressWarnings("unchecked")
    public T get(int index) {
        return (T) array[index];  // 需要强制转换
    }
}
```

## 六、泛型约束

### 6.1 边界约束

```java
// T必须是Number的子类
public class NumericBox<T extends Number> {
    private T value;
    
    public double doubleValue() {
        return value.doubleValue();  // 可以调用Number的方法
    }
}

// 多个边界
public class MultiBound<T extends Comparable<T> & Serializable> {
    private T value;
}
```

### 6.2 常见约束

```java
// 可比较
<T extends Comparable<T>>

// 可序列化
<T extends Serializable>

// 特定接口
<T extends Runnable>
```

## 七、实战案例

### 7.1 通用工具类

```java
public class CollectionUtils {
    
    // 查找最大值
    public static <T extends Comparable<T>> T findMax(List<T> list) {
        if (list == null || list.isEmpty()) {
            return null;
        }
        
        T max = list.get(0);
        for (T item : list) {
            if (item.compareTo(max) > 0) {
                max = item;
            }
        }
        return max;
    }
    
    // 交换列表中的两个元素
    public static <T> void swap(List<T> list, int i, int j) {
        T temp = list.get(i);
        list.set(i, list.get(j));
        list.set(j, temp);
    }
}
```

### 7.2 泛型缓存

```java
public class Cache<K, V> {
    private Map<K, V> cache = new HashMap<>();
    
    public void put(K key, V value) {
        cache.put(key, value);
    }
    
    public V get(K key) {
        return cache.get(key);
    }
    
    public boolean containsKey(K key) {
        return cache.containsKey(key);
    }
}
```
