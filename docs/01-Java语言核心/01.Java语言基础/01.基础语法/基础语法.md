# Java 基础语法

## 一、Java 程序结构

### 1.1 Hello World 程序

```java
public class HelloWorld {
    public static void main(String[] args) {
        System.out.println("Hello, World!");
    }
}
```

**代码解析：**
- `public`：访问修饰符，表示该类是公开的
- `class`：定义类的关键字
- `HelloWorld`：类名，必须与文件名相同
- `main`：程序的入口方法
- `String[] args`：命令行参数
- `System.out.println()`：输出语句

### 1.2 Java 程序执行流程

```
.java 源文件 → javac 编译 → .class 字节码文件 → JVM 解释执行
```

## 二、数据类型

### 2.1 基本数据类型（8种）

#### 整数类型

| 类型 | 占用空间 | 取值范围 | 默认值 |
|------|---------|---------|--------|
| byte | 1字节 | -128 ~ 127 | 0 |
| short | 2字节 | -32768 ~ 32767 | 0 |
| int | 4字节 | -2^31 ~ 2^31-1 | 0 |
| long | 8字节 | -2^63 ~ 2^63-1 | 0L |

```java
byte b = 100;
short s = 10000;
int i = 100000;
long l = 100000L;  // long类型需要加L后缀
```

#### 浮点类型

| 类型 | 占用空间 | 精度 | 默认值 |
|------|---------|------|--------|
| float | 4字节 | 7位有效数字 | 0.0f |
| double | 8字节 | 15位有效数字 | 0.0d |

```java
float f = 3.14f;    // float类型需要加f后缀
double d = 3.14159; // double是默认浮点类型
```

#### 字符类型

| 类型 | 占用空间 | 取值范围 | 默认值 |
|------|---------|---------|--------|
| char | 2字节 | 0 ~ 65535 | '\u0000' |

```java
char c1 = 'A';
char c2 = 65;      // ASCII码
char c3 = '\u0041'; // Unicode编码
```

#### 布尔类型

| 类型 | 取值 | 默认值 |
|------|------|--------|
| boolean | true/false | false |

```java
boolean flag = true;
boolean isReady = false;
```

### 2.2 引用数据类型

- 类（Class）
- 接口（Interface）
- 数组（Array）
- 字符串（String）

```java
String str = "Hello";
int[] arr = {1, 2, 3};
Object obj = new Object();
```

### 2.3 数据类型转换

#### 自动类型转换（隐式转换）

小类型 → 大类型，自动完成

```java
int i = 100;
long l = i;      // int自动转换为long
double d = l;    // long自动转换为double
```

**转换规则：**
```
byte → short → int → long → float → double
                ↑
               char
```

#### 强制类型转换（显式转换）

大类型 → 小类型，需要强制转换，可能丢失精度

```java
double d = 3.14;
int i = (int)d;  // i的值为3，小数部分丢失

long l = 1000;
int i2 = (int)l; // 需要强制转换
```

**注意事项：**
- 可能导致精度丢失
- 可能溢出
- boolean类型不能与其他类型转换

## 三、变量与常量

### 3.1 变量

#### 变量声明与初始化

```java
// 先声明后赋值
int age;
age = 25;

// 声明同时赋值
int score = 90;

// 多个变量同时声明
int x = 1, y = 2, z = 3;
```

#### 变量分类

| 类型 | 位置 | 生命周期 | 默认值 |
|------|------|---------|--------|
| 局部变量 | 方法内部 | 方法执行期间 | 无，必须初始化 |
| 实例变量 | 类中方法外 | 对象存在期间 | 有默认值 |
| 静态变量 | 类中方法外，用static修饰 | 类加载到卸载 | 有默认值 |

```java
public class VariableDemo {
    static int staticVar = 10;    // 静态变量
    int instanceVar = 20;         // 实例变量
    
    public void method() {
        int localVar = 30;        // 局部变量，必须初始化
        System.out.println(localVar);
    }
}
```

### 3.2 常量

使用 `final` 关键字修饰，一旦赋值不可更改

```java
final double PI = 3.14159;
final int MAX_SIZE = 100;

// 编译错误：无法为最终变量PI分配值
// PI = 3.14;
```

**命名规范：** 常量名全部大写，单词间用下划线分隔

## 四、运算符

### 4.1 算术运算符

| 运算符 | 说明 | 示例 | 结果 |
|--------|------|------|------|
| + | 加法 | 5 + 3 | 8 |
| - | 减法 | 5 - 3 | 2 |
| * | 乘法 | 5 * 3 | 15 |
| / | 除法 | 5 / 2 | 2 |
| % | 取模（取余） | 5 % 2 | 1 |
| ++ | 自增 | i++ | i = i + 1 |
| -- | 自减 | i-- | i = i - 1 |

```java
int a = 10;
int b = 3;

System.out.println(a + b);  // 13
System.out.println(a - b);  // 7
System.out.println(a * b);  // 30
System.out.println(a / b);  // 3（整数除法）
System.out.println(a % b);  // 1

// 自增自减
int i = 5;
System.out.println(i++);  // 5（先使用后增加）
System.out.println(i);    // 6
System.out.println(++i);  // 7（先增加后使用）
```

### 4.2 赋值运算符

| 运算符 | 说明 | 示例 | 等价于 |
|--------|------|------|--------|
| = | 简单赋值 | a = 5 | a = 5 |
| += | 加后赋值 | a += 3 | a = a + 3 |
| -= | 减后赋值 | a -= 3 | a = a - 3 |
| *= | 乘后赋值 | a *= 3 | a = a * 3 |
| /= | 除后赋值 | a /= 3 | a = a / 3 |
| %= | 取模后赋值 | a %= 3 | a = a % 3 |

### 4.3 比较运算符

结果为 boolean 类型

| 运算符 | 说明 | 示例 | 结果 |
|--------|------|------|------|
| == | 等于 | 5 == 3 | false |
| != | 不等于 | 5 != 3 | true |
| > | 大于 | 5 > 3 | true |
| < | 小于 | 5 < 3 | false |
| >= | 大于等于 | 5 >= 5 | true |
| <= | 小于等于 | 5 <= 3 | false |

### 4.4 逻辑运算符

| 运算符 | 说明 | 示例 | 结果 |
|--------|------|------|------|
| && | 短路与 | true && false | false |
| \|\| | 短路或 | true \|\| false | true |
| ! | 非 | !true | false |
| & | 逻辑与 | true & false | false |
| \| | 逻辑或 | true \| false | true |

```java
boolean a = true;
boolean b = false;

System.out.println(a && b);  // false
System.out.println(a || b);  // true
System.out.println(!a);      // false

// 短路与 vs 逻辑与
int x = 5;
boolean result = (x > 3) && (x++ < 10);  // x不会自增，因为前面已确定为false
```

### 4.5 三元运算符

```java
// 条件 ? 表达式1 : 表达式2
int a = 10;
int b = 20;
int max = (a > b) ? a : b;  // max = 20

// 判断奇偶
int num = 7;
String type = (num % 2 == 0) ? "偶数" : "奇数";
```

### 4.6 位运算符

| 运算符 | 说明 | 示例 | 结果 |
|--------|------|------|------|
| & | 按位与 | 5 & 3 | 1 |
| \| | 按位或 | 5 \| 3 | 7 |
| ^ | 按位异或 | 5 ^ 3 | 6 |
| ~ | 按位取反 | ~5 | -6 |
| << | 左移 | 5 << 1 | 10 |
| >> | 右移 | 5 >> 1 | 2 |
| >>> | 无符号右移 | -5 >>> 1 | 2147483645 |

```java
// 位运算示例
System.out.println(5 & 3);   // 1  (0101 & 0011 = 0001)
System.out.println(5 | 3);   // 7  (0101 | 0011 = 0111)
System.out.println(5 ^ 3);   // 6  (0101 ^ 0011 = 0110)
System.out.println(5 << 1);  // 10 (左移1位相当于乘以2)
System.out.println(5 >> 1);  // 2  (右移1位相当于除以2)
```

## 五、流程控制

### 5.1 顺序结构

代码从上到下依次执行

```java
int a = 10;
int b = 20;
int sum = a + b;
System.out.println(sum);  // 30
```

### 5.2 分支结构

#### if-else 语句

```java
// 单分支
if (score >= 60) {
    System.out.println("及格");
}

// 双分支
if (score >= 60) {
    System.out.println("及格");
} else {
    System.out.println("不及格");
}

// 多分支
if (score >= 90) {
    System.out.println("优秀");
} else if (score >= 80) {
    System.out.println("良好");
} else if (score >= 60) {
    System.out.println("及格");
} else {
    System.out.println("不及格");
}
```

#### switch-case 语句

```java
int day = 3;
switch (day) {
    case 1:
        System.out.println("星期一");
        break;
    case 2:
        System.out.println("星期二");
        break;
    case 3:
        System.out.println("星期三");
        break;
    default:
        System.out.println("其他");
        break;
}

// JDK 12+ 新语法
String result = switch (day) {
    case 1 -> "星期一";
    case 2 -> "星期二";
    case 3 -> "星期三";
    default -> "其他";
};
```

**注意事项：**
- case 后面必须是常量
- 不要忘记 break，否则会穿透
- default 可以省略

### 5.3 循环结构

#### for 循环

```java
// 基本for循环
for (int i = 0; i < 5; i++) {
    System.out.println(i);  // 0 1 2 3 4
}

// 增强for循环（foreach）
int[] arr = {1, 2, 3, 4, 5};
for (int num : arr) {
    System.out.println(num);
}
```

#### while 循环

```java
int i = 0;
while (i < 5) {
    System.out.println(i);
    i++;
}
```

#### do-while 循环

至少执行一次

```java
int i = 0;
do {
    System.out.println(i);
    i++;
} while (i < 5);
```

#### 循环控制语句

```java
// break：跳出循环
for (int i = 0; i < 10; i++) {
    if (i == 5) {
        break;  // 当i=5时跳出循环
    }
    System.out.println(i);  // 0 1 2 3 4
}

// continue：跳过本次循环
for (int i = 0; i < 5; i++) {
    if (i == 2) {
        continue;  // 跳过i=2的这次循环
    }
    System.out.println(i);  // 0 1 3 4
}
```

## 六、数组

### 6.1 数组声明与初始化

```java
// 方式1：先声明后初始化
int[] arr1;
arr1 = new int[5];

// 方式2：声明同时初始化
int[] arr2 = new int[5];

// 方式3：直接赋值
int[] arr3 = {1, 2, 3, 4, 5};

// 方式4：new的方式赋值
int[] arr4 = new int[]{1, 2, 3, 4, 5};
```

### 6.2 数组访问

```java
int[] arr = {10, 20, 30, 40, 50};

// 访问元素
System.out.println(arr[0]);  // 10
System.out.println(arr[4]);  // 50

// 修改元素
arr[0] = 100;

// 数组长度
System.out.println(arr.length);  // 5

// 遍历数组
for (int i = 0; i < arr.length; i++) {
    System.out.println(arr[i]);
}
```

### 6.3 多维数组

```java
// 二维数组
int[][] matrix = {
    {1, 2, 3},
    {4, 5, 6},
    {7, 8, 9}
};

// 访问二维数组
System.out.println(matrix[0][0]);  // 1
System.out.println(matrix[1][2]);  // 6

// 遍历二维数组
for (int i = 0; i < matrix.length; i++) {
    for (int j = 0; j < matrix[i].length; j++) {
        System.out.print(matrix[i][j] + " ");
    }
    System.out.println();
}
```

### 6.4 数组常用操作

```java
import java.util.Arrays;

int[] arr = {5, 2, 8, 1, 9};

// 排序
Arrays.sort(arr);  // [1, 2, 5, 8, 9]

// 查找（二分查找，需要先排序）
int index = Arrays.binarySearch(arr, 5);

// 填充
Arrays.fill(arr, 0);  // [0, 0, 0, 0, 0]

// 复制
int[] copy = Arrays.copyOf(arr, arr.length);

// 转字符串
System.out.println(Arrays.toString(arr));
```

## 七、键盘输入

### 7.1 Scanner 类

```java
import java.util.Scanner;

public class InputDemo {
    public static void main(String[] args) {
        Scanner scanner = new Scanner(System.in);
        
        // 读取整数
        System.out.print("请输入年龄：");
        int age = scanner.nextInt();
        
        // 读取浮点数
        System.out.print("请输入身高：");
        double height = scanner.nextDouble();
        
        // 读取字符串
        System.out.print("请输入姓名：");
        String name = scanner.next();        // 读取一个单词
        String line = scanner.nextLine();    // 读取一行
        
        // 读取布尔值
        System.out.print("是否继续（true/false）：");
        boolean flag = scanner.nextBoolean();
        
        scanner.close();  // 关闭scanner
    }
}
```

### 7.2 注意事项

```java
Scanner scanner = new Scanner(System.in);

// nextInt()后接nextLine()的问题
int num = scanner.nextInt();
scanner.nextLine();  // 消耗掉回车符
String str = scanner.nextLine();
```

## 八、方法（函数）

### 8.1 方法定义

```java
修饰符 返回值类型 方法名(参数列表) {
    // 方法体
    return 返回值;
}
```

**示例：**

```java
public class MethodDemo {
    
    // 无参无返回值
    public void sayHello() {
        System.out.println("Hello!");
    }
    
    // 有参无返回值
    public void printName(String name) {
        System.out.println("姓名：" + name);
    }
    
    // 有参有返回值
    public int add(int a, int b) {
        return a + b;
    }
    
    // 无参有返回值
    public String getGreeting() {
        return "Good Morning!";
    }
}
```

### 8.2 方法调用

```java
MethodDemo demo = new MethodDemo();

demo.sayHello();              // 调用无参方法
demo.printName("张三");        // 调用有参方法
int sum = demo.add(10, 20);   // 接收返回值
String greeting = demo.getGreeting();
```

### 8.3 方法重载

同一个类中，方法名相同，参数列表不同

```java
public class OverloadDemo {
    
    public int add(int a, int b) {
        return a + b;
    }
    
    public double add(double a, double b) {
        return a + b;
    }
    
    public int add(int a, int b, int c) {
        return a + b + c;
    }
}
```

**重载条件：**
- 方法名相同
- 参数个数不同 或 参数类型不同 或 参数顺序不同
- 与返回值类型无关

### 8.4 可变参数

```java
public class VarArgsDemo {
    
    // 可变参数必须是最后一个参数
    public int sum(int... numbers) {
        int total = 0;
        for (int num : numbers) {
            total += num;
        }
        return total;
    }
}

// 调用
sum(1, 2, 3);           // 6
sum(1, 2, 3, 4, 5);     // 15
sum();                  // 0
```

## 十、常见错误与注意事项

### 10.1 常见编译错误

1. **缺少分号**
   ```java
   int a = 10  // 错误：缺少分号
   int a = 10; // 正确
   ```

2. **括号不匹配**
   ```java
   if (a > b) {
       System.out.println(a);
   // 错误：缺少右括号
   ```

3. **变量未初始化**
   ```java
   int a;
   System.out.println(a);  // 错误：局部变量必须初始化
   ```

4. **类型不匹配**
   ```java
   int a = 3.14;  // 错误：不能将double赋值给int
   int a = (int)3.14;  // 正确：需要强制转换
   ```

### 10.2 常见运行错误

1. **数组越界**
   ```java
   int[] arr = new int[5];
   System.out.println(arr[5]);  // ArrayIndexOutOfBoundsException
   ```

2. **空指针异常**
   ```java
   String str = null;
   System.out.println(str.length());  // NullPointerException
   ```

3. **算术异常**
   ```java
   int a = 10 / 0;  // ArithmeticException
   ```
