# 集合框架

## 一、集合概述

### 1.1 什么是集合？

集合是 Java 提供的一种容器，用于存储和操作一组对象。

**集合 vs 数组：**

| 对比项 | 数组 | 集合 |
|--------|------|------|
| 长度 | 固定 | 可变 |
| 类型 | 可存基本类型 | 只能存引用类型 |
| 功能 | 简单 | 丰富（增删改查） |
| 性能 | 高 | 相对较低 |

### 1.2 集合体系结构

```
Collection（接口）
├── List（接口）- 有序、可重复
│   ├── ArrayList
│   ├── LinkedList
│   └── Vector
│
├── Set（接口）- 无序、不可重复
│   ├── HashSet
│   ├── LinkedHashSet
│   └── TreeSet
│
└── Queue（接口）- 队列
    ├── PriorityQueue
    └── ArrayDeque

Map（接口）- 键值对
├── HashMap
├── LinkedHashMap
├── TreeMap
├── Hashtable
└── ConcurrentHashMap
```

## 三、List 接口

### 3.1 List 特点

- **有序**：元素按插入顺序存储
- **可重复**：允许重复元素
- **可通过索引访问**：支持随机访问

### 3.2 ArrayList

#### 基本使用

```java
import java.util.ArrayList;
import java.util.List;

List<String> list = new ArrayList<>();

// 添加元素
list.add("Apple");
list.add("Banana");
list.add("Cherry");

// 指定位置插入
list.add(1, "Orange");

// 获取元素
String fruit = list.get(0);  // "Apple"

// 修改元素
list.set(0, "Grape");

// 删除元素
list.remove(0);           // 按索引删除
list.remove("Banana");    // 按对象删除

// 查找元素
int index = list.indexOf("Cherry");  // 返回索引，不存在返回-1

// 截取子列表
List<String> subList = list.subList(0, 2);

// 遍历
for (int i = 0; i < list.size(); i++) {
    System.out.println(list.get(i));
}

for (String item : list) {
    System.out.println(item);
}
```

#### 底层原理

- **数据结构**：动态数组
- **初始容量**：10（JDK 8+）
- **扩容机制**：1.5倍扩容
- **适用场景**：频繁查询，少量增删

```java
// 指定初始容量
List<String> list = new ArrayList<>(100);
```

### 3.3 LinkedList

```java
import java.util.LinkedList;

LinkedList<String> linkedList = new LinkedList<>();

// 添加到头部/尾部
linkedList.addFirst("First");
linkedList.addLast("Last");

// 获取头部/尾部
String first = linkedList.getFirst();
String last = linkedList.getLast();

// 删除头部/尾部
linkedList.removeFirst();
linkedList.removeLast();

// 作为栈使用
linkedList.push("Element");
String popped = linkedList.pop();

// 作为队列使用
linkedList.offer("Element");
String polled = linkedList.poll();
```

#### 底层原理

- **数据结构**：双向链表
- **适用场景**：频繁增删，少量查询

### 3.4 ArrayList vs LinkedList

| 对比项 | ArrayList | LinkedList |
|--------|-----------|------------|
| 底层结构 | 数组 | 链表 |
| 随机访问 | O(1) | O(n) |
| 头部插入 | O(n) | O(1) |
| 尾部插入 | O(1)* | O(1) |
| 中间插入 | O(n) | O(n) |
| 内存占用 | 少 | 多（节点指针） |
| 适用场景 | 查询多 | 增删多 |

*\*ArrayList 尾部插入平均 O(1)，扩容时 O(n)*

## 四、Set 接口

### 4.1 Set 特点

- **无序**：不保证元素顺序（LinkedHashSet 除外）
- **不可重复**：不允许重复元素
- **最多一个 null 值**

### 4.2 HashSet

```java
import java.util.HashSet;
import java.util.Set;

Set<String> set = new HashSet<>();

// 添加元素
set.add("Apple");
set.add("Banana");
set.add("Apple");  // 重复，添加失败

// 判断是否包含
boolean contains = set.contains("Apple");

// 删除元素
set.remove("Banana");

// 遍历
for (String item : set) {
    System.out.println(item);
}

// 大小
int size = set.size();
```

#### 底层原理

- **数据结构**：HashMap
- **去重原理**：hashCode() + equals()
- **性能**：添加、删除、查找都是 O(1)

### 4.3 LinkedHashSet

```java
import java.util.LinkedHashSet;

// 保持插入顺序
LinkedHashSet<String> linkedSet = new LinkedHashSet<>();
linkedSet.add("C");
linkedSet.add("A");
linkedSet.add("B");

// 遍历顺序：C, A, B（插入顺序）
for (String item : linkedSet) {
    System.out.println(item);
}
```

### 4.4 TreeSet

```java
import java.util.TreeSet;

// 自然排序
TreeSet<Integer> treeSet = new TreeSet<>();
treeSet.add(3);
treeSet.add(1);
treeSet.add(2);

// 遍历顺序：1, 2, 3（升序）
for (Integer num : treeSet) {
    System.out.println(num);
}

// 自定义排序
TreeSet<String> sortedSet = new TreeSet<>((a, b) -> b.compareTo(a));
sortedSet.add("C");
sortedSet.add("A");
sortedSet.add("B");

// 遍历顺序：C, B, A（降序）
```

#### 底层原理

- **数据结构**：红黑树
- **性能**：添加、删除、查找都是 O(log n)
- **特点**：元素有序

### 4.5 Set 去重原理

```java
public class Person {
    private String name;
    private int age;
    
    @Override
    public int hashCode() {
        return Objects.hash(name, age);
    }
    
    @Override
    public boolean equals(Object obj) {
        if (this == obj) return true;
        if (obj == null || getClass() != obj.getClass()) return false;
        Person person = (Person) obj;
        return age == person.age && Objects.equals(name, person.name);
    }
}

// 使用
Set<Person> set = new HashSet<>();
set.add(new Person("张三", 20));
set.add(new Person("张三", 20));  // 重复，不会添加
System.out.println(set.size());  // 1
```

**去重步骤：**
1. 计算 hashCode，确定桶位置
2. 如果桶为空，直接添加
3. 如果桶不为空，遍历链表/红黑树
4. 调用 equals 比较，如果相等则不添加

## 五、Queue 接口

### 5.1 Queue 特点

- **先进先出（FIFO）**
- 支持在尾部添加，头部删除

### 5.2 PriorityQueue（优先队列）

```java
import java.util.PriorityQueue;

// 自然排序（最小堆）
PriorityQueue<Integer> pq = new PriorityQueue<>();
pq.offer(3);
pq.offer(1);
pq.offer(2);

System.out.println(pq.poll());  // 1（最小值）
System.out.println(pq.poll());  // 2
System.out.println(pq.poll());  // 3

// 自定义排序（最大堆）
PriorityQueue<Integer> maxHeap = new PriorityQueue<>((a, b) -> b - a);
maxHeap.offer(3);
maxHeap.offer(1);
maxHeap.offer(2);

System.out.println(maxHeap.poll());  // 3（最大值）
```

### 5.3 ArrayDeque（双端队列）

```java
import java.util.ArrayDeque;

ArrayDeque<String> deque = new ArrayDeque<>();

// 添加到头部/尾部
deque.addFirst("First");
deque.addLast("Last");

// 从头部/尾部删除
String first = deque.removeFirst();
String last = deque.removeLast();

// 作为栈使用
deque.push("Element");
String popped = deque.pop();

// 作为队列使用
deque.offer("Element");
String polled = deque.poll();
```

## 六、Map 接口

### 6.1 Map 特点

- **键值对存储**
- **键唯一**，值可重复
- **键最多一个 null**

### 6.2 HashMap

```java
import java.util.HashMap;
import java.util.Map;

Map<String, Integer> map = new HashMap<>();

// 添加元素
map.put("Apple", 5);
map.put("Banana", 3);

// 获取元素
Integer count = map.get("Apple");  // 5

// 判断是否包含
boolean hasKey = map.containsKey("Apple");
boolean hasValue = map.containsValue(5);

// 删除元素
map.remove("Banana");

// 修改元素
map.put("Apple", 10);  // 覆盖原值

// 大小
int size = map.size();

// 遍历方式1：keySet
for (String key : map.keySet()) {
    System.out.println(key + ": " + map.get(key));
}

// 遍历方式2：entrySet（推荐）
for (Map.Entry<String, Integer> entry : map.entrySet()) {
    System.out.println(entry.getKey() + ": " + entry.getValue());
}

// 遍历方式3：forEach（JDK 8+）
map.forEach((key, value) -> {
    System.out.println(key + ": " + value);
});
```

#### 底层原理

- **数据结构**：数组 + 链表 + 红黑树（JDK 8+）
- **初始容量**：16
- **负载因子**：0.75
- **扩容阈值**：容量 * 负载因子
- **树化阈值**：链表长度 > 8 且数组长度 >= 64

```java
// 指定初始容量和负载因子
Map<String, Integer> map = new HashMap<>(32, 0.75f);
```

### 6.3 LinkedHashMap

```java
import java.util.LinkedHashMap;

// 保持插入顺序
LinkedHashMap<String, Integer> linkedMap = new LinkedHashMap<>();
linkedMap.put("C", 3);
linkedMap.put("A", 1);
linkedMap.put("B", 2);

// 遍历顺序：C, A, B（插入顺序）
for (Map.Entry<String, Integer> entry : linkedMap.entrySet()) {
    System.out.println(entry.getKey() + ": " + entry.getValue());
}

// 访问顺序（LRU缓存实现）
LinkedHashMap<String, Integer> accessOrderMap = 
    new LinkedHashMap<>(16, 0.75f, true);
```

### 6.4 TreeMap

```java
import java.util.TreeMap;

// 自然排序
TreeMap<Integer, String> treeMap = new TreeMap<>();
treeMap.put(3, "Three");
treeMap.put(1, "One");
treeMap.put(2, "Two");

// 遍历顺序：1, 2, 3（按键升序）
for (Map.Entry<Integer, String> entry : treeMap.entrySet()) {
    System.out.println(entry.getKey() + ": " + entry.getValue());
}

// 自定义排序
TreeMap<String, Integer> sortedMap = 
    new TreeMap<>((a, b) -> b.compareTo(a));

// 范围查询
TreeMap<Integer, String> rangeMap = new TreeMap<>();
rangeMap.put(1, "One");
rangeMap.put(2, "Two");
rangeMap.put(3, "Three");
rangeMap.put(4, "Four");

Map<Integer, String> subMap = rangeMap.subMap(2, 4);  // [2, 4)
```

### 6.5 Hashtable（已过时）

```java
import java.util.Hashtable;

// Hashtable 是线程安全的 HashMap（已过时）
Hashtable<String, Integer> table = new Hashtable<>();
table.put("Key", 1);

// 推荐使用 ConcurrentHashMap 替代
Map<String, Integer> concurrentMap = new ConcurrentHashMap<>();
```

### 6.6 ConcurrentHashMap

```java
import java.util.concurrent.ConcurrentHashMap;

// 线程安全的 HashMap
ConcurrentHashMap<String, Integer> concurrentMap = new ConcurrentHashMap<>();

// 线程安全的操作
concurrentMap.putIfAbsent("Key", 1);
concurrentMap.computeIfAbsent("Key", k -> 0);
concurrentMap.merge("Key", 1, Integer::sum);

// 分段锁，性能优于 Hashtable
```

### 6.7 Map 对比

| 对比项 | HashMap | LinkedHashMap | TreeMap | ConcurrentHashMap |
|--------|---------|---------------|---------|-------------------|
| 有序性 | 无序 | 插入顺序 | 键排序 | 无序 |
| 线程安全 | 否 | 否 | 否 | 是 |
| null键/值 | 允许 | 允许 | 不允许 | 不允许 |
| 性能 | 高 | 中 | 低 | 高 |
| 底层结构 | 哈希表 | 哈希表+链表 | 红黑树 | 分段哈希表 |

## 八、迭代器

### 8.1 Iterator

```java
List<String> list = new ArrayList<>();

    String item = iterator.next();

    // 安全删除
```

```java
    String item = listIterator.next();
    System.out.println(item);
```

## 九、Comparable 和 Comparator

### 9.1 Comparable（自然排序）

```java
public class Student implements Comparable<Student> {
    private String name;
    private int age;
    
    @Override
    public int compareTo(Student other) {
        return this.age - other.age;  // 按年龄升序
    }
}

// 使用
List<Student> students = new ArrayList<>();
students.add(new Student("张三", 20));
students.add(new Student("李四", 18));

Collections.sort(students);  // 按年龄排序
```

### 9.2 Comparator（定制排序）

```java
List<Student> students = new ArrayList<>();

// Lambda表达式
students.sort((s1, s2) -> s1.getName().compareTo(s2.getName()));

// 方法引用
students.sort(Comparator.comparing(Student::getName));

// 链式排序
students.sort(Comparator.comparing(Student::getAge)
                        .thenComparing(Student::getName));

// 降序
students.sort(Comparator.comparing(Student::getAge).reversed());
```

## 十、实战案例

### 10.1 LRU 缓存

```java
public class LRUCache<K, V> extends LinkedHashMap<K, V> {
    private final int capacity;
    
    public LRUCache(int capacity) {
        super(capacity, 0.75f, true);  // 访问顺序
        this.capacity = capacity;
    }
    
    @Override
    protected boolean removeEldestEntry(Map.Entry<K, V> eldest) {
        return size() > capacity;
    }
}

// 使用
LRUCache<String, Integer> cache = new LRUCache<>(3);
cache.put("A", 1);
cache.put("B", 2);
cache.put("C", 3);
cache.get("A");  // 访问A
cache.put("D", 4);  // B被移除（最久未使用）
```

### 10.2 统计词频

```java
public Map<String, Integer> countWords(List<String> words) {
    Map<String, Integer> wordCount = new HashMap<>();
    
    for (String word : words) {
        wordCount.merge(word.toLowerCase(), 1, Integer::sum);
    }
    
    return wordCount;
}
```

### 10.3 分组统计

```java
public Map<String, List<Student>> groupByClass(List<Student> students) {
    Map<String, List<Student>> grouped = new HashMap<>();
    
    for (Student student : students) {
        grouped.computeIfAbsent(student.getClassName(), 
            k -> new ArrayList<>()).add(student);
    }
    
    return grouped;
}
```

## 十一、性能优化建议

### 11.1 选择合适的集合

| 场景 | 推荐集合 |
|------|---------|
| 频繁查询 | ArrayList |
| 频繁增删 | LinkedList |
| 去重 | HashSet |
| 有序去重 | TreeSet |
| 键值对 | HashMap |
| 有序键值对 | TreeMap |
| 线程安全 | ConcurrentHashMap |
| LRU缓存 | LinkedHashMap |

### 11.2 初始化容量

```java
// 不好：频繁扩容
List<String> list = new ArrayList<>();
for (int i = 0; i < 10000; i++) {
    list.add(String.valueOf(i));
}

// 好：指定初始容量
List<String> list = new ArrayList<>(10000);
for (int i = 0; i < 10000; i++) {
    list.add(String.valueOf(i));
}
```

### 11.3 避免装箱拆箱

```java
// 不好：自动装箱
List<Integer> list = new ArrayList<>();
for (int i = 0; i < 1000000; i++) {
    list.add(i);  // 每次都要装箱
}

// 好：使用基本类型数组
int[] array = new int[1000000];
for (int i = 0; i < 1000000; i++) {
    array[i] = i;
}
```
