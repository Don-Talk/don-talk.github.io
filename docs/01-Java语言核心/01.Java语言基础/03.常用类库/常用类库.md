# 常用类库

## 一、String 类

### 1.1 String 概述

String 是 Java 中最常用的类，用于表示字符串。

**特点：**
- 不可变性：一旦创建，内容不能修改
- 存储在常量池（JDK 7+）或堆中
- 实现了 `Serializable`、`Comparable`、`CharSequence` 接口

### 1.2 String 创建方式

```java
// 方式1：直接赋值（推荐）
String str1 = "Hello";

// 方式2：new 的方式
String str2 = new String("Hello");

// 方式3：字符数组
char[] chars = {'H', 'e', 'l', 'l', 'o'};
String str3 = new String(chars);

// 方式4：字节数组
byte[] bytes = {72, 101, 108, 108, 111};
String str4 = new String(bytes);
```

**区别：**
```java
String s1 = "Hello";
String s2 = "Hello";
String s3 = new String("Hello");

System.out.println(s1 == s2);   // true（指向常量池同一对象）
System.out.println(s1 == s3);   // false（s3在堆中）
System.out.println(s1.equals(s3)); // true（内容相同）
```

### 1.3 常用方法

#### 获取功能

```java
String str = "Hello World";

// 长度
int len = str.length();  // 11

// 获取指定位置字符
char c = str.charAt(0);  // 'H'

// 查找子串位置
int index = str.indexOf("World");    // 6
int lastIndex = str.lastIndexOf("o"); // 7

// 截取子串
String sub1 = str.substring(6);      // "World"
String sub2 = str.substring(0, 5);   // "Hello"
```

#### 判断功能

```java
String str = "Hello World";

// 是否包含
boolean contains = str.contains("World");  // true

// 是否为空
boolean empty = str.isEmpty();             // false

// 是否以某字符串开头/结尾
boolean startsWith = str.startsWith("Hello"); // true
boolean endsWith = str.endsWith("World");     // true

// 比较
boolean equals = str.equals("Hello World");   // true
boolean equalsIgnoreCase = str.equalsIgnoreCase("hello world"); // true
```

#### 转换功能

```java
String str = "Hello World";

// 转大写/小写
String upper = str.toUpperCase();    // "HELLO WORLD"
String lower = str.toLowerCase();    // "hello world"

// 转字符数组
char[] chars = str.toCharArray();

// 转字节数组
byte[] bytes = str.getBytes();

// 替换
String replaced = str.replace("World", "Java");  // "Hello Java"

// 分割
String[] parts = "a,b,c".split(",");  // ["a", "b", "c"]

// 去除首尾空格
String trimmed = "  Hello  ".trim();  // "Hello"
```

#### 其他方法

```java
// 拼接
String str1 = "Hello";
String str2 = "World";
String result = str1.concat(str2);  // "HelloWorld"
String result2 = str1 + " " + str2; // "Hello World"

// intern()：将字符串放入常量池
String s1 = new String("Hello");
String s2 = s1.intern();
String s3 = "Hello";
System.out.println(s2 == s3);  // true
```

### 1.4 String 不可变性

```java
String str = "Hello";
str = str + " World";  // 创建了新的字符串对象

// 原理解释：
// 1. 创建 "Hello" 对象
// 2. 创建 " World" 对象
// 3. 创建 "Hello World" 对象
// 4. str 引用指向新对象
```

**优点：**
- 线程安全
- 可以缓存哈希值
- 安全性高（如数据库连接URL、用户名密码等）

## 二、StringBuilder 和 StringBuffer

### 2.1 为什么需要 StringBuilder？

String 是不可变的，频繁拼接会产生大量临时对象，影响性能。

```java
// 低效方式
String str = "";
for (int i = 0; i < 10000; i++) {
    str += i;  // 每次都创建新对象
}

// 高效方式
StringBuilder sb = new StringBuilder();
for (int i = 0; i < 10000; i++) {
    sb.append(i);
}
String result = sb.toString();
```

### 2.2 StringBuilder vs StringBuffer

| 对比项 | StringBuilder | StringBuffer |
|--------|--------------|--------------|
| 线程安全 | 否 | 是 |
| 性能 | 快 | 慢 |
| 适用场景 | 单线程 | 多线程 |

### 2.3 常用方法

```java
StringBuilder sb = new StringBuilder();

// 追加
sb.append("Hello");
sb.append(" World");

// 插入
sb.insert(5, ",");  // "Hello, World"

// 删除
sb.delete(5, 6);    // "Hello World"

// 替换
sb.replace(0, 5, "Hi");  // "Hi World"

// 反转
sb.reverse();  // "dlroW iH"

// 长度
int len = sb.length();

// 转String
String str = sb.toString();
```

### 2.4 链式调用

```java
StringBuilder sb = new StringBuilder()
    .append("Hello")
    .append(" ")
    .append("World")
    .append("!");

System.out.println(sb.toString());  // "Hello World!"
```

## 三、包装类

### 3.1 什么是包装类？

包装类是将基本数据类型封装成对象的类。

| 基本类型 | 包装类 |
|---------|--------|
| byte | Byte |
| short | Short |
| int | Integer |
| long | Long |
| float | Float |
| double | Double |
| char | Character |
| boolean | Boolean |

### 3.2 装箱与拆箱

```java
// 自动装箱：基本类型 → 包装类
Integer num1 = 100;

// 自动拆箱：包装类 → 基本类型
int num2 = num1;

// 手动装箱
Integer num3 = Integer.valueOf(100);

// 手动拆箱
int num4 = num3.intValue();
```

### 3.3 常用方法

```java
// String 转基本类型
int num1 = Integer.parseInt("123");
double num2 = Double.parseDouble("3.14");
boolean flag = Boolean.parseBoolean("true");

// 基本类型转String
String str1 = String.valueOf(123);
String str2 = Integer.toString(123);

// 最大值最小值
int max = Integer.MAX_VALUE;
int min = Integer.MIN_VALUE;

// 比较
Integer a = 100;
Integer b = 200;
int result = a.compareTo(b);  // -1（小于）
```

### 3.4 缓存机制

```java
// Integer 缓存范围：-128 ~ 127
Integer a = 100;
Integer b = 100;
System.out.println(a == b);  // true（使用缓存）

Integer c = 200;
Integer d = 200;
System.out.println(c == d);  // false（超出缓存范围）

// 注意：比较Integer应该用equals
System.out.println(c.equals(d));  // true
```

## 四、Math 类

### 4.1 常用方法

```java
// 绝对值
int abs = Math.abs(-10);  // 10

// 最大值/最小值
int max = Math.max(10, 20);   // 20
int min = Math.min(10, 20);   // 10

// 向上/向下取整
double ceil = Math.ceil(3.14);   // 4.0
double floor = Math.floor(3.14); // 3.0
long round = Math.round(3.5);    // 4

// 幂运算
double pow = Math.pow(2, 3);  // 8.0

// 平方根
double sqrt = Math.sqrt(16);  // 4.0

// 随机数
double random = Math.random();  // [0.0, 1.0)

// 三角函数
double sin = Math.sin(Math.PI / 2);  // 1.0
double cos = Math.cos(0);            // 1.0

// 常量
double pi = Math.PI;   // 3.141592653589793
double e = Math.E;     // 2.718281828459045
```

### 4.2 生成随机数

```java
// 方式1：Math.random()
int num1 = (int)(Math.random() * 100);  // 0-99

// 方式2：Random类
import java.util.Random;
Random random = new Random();
int num2 = random.nextInt(100);  // 0-99
int num3 = random.nextInt(50, 100);  // 50-99 (JDK 17+)

// 方式3：ThreadLocalRandom（推荐，线程安全）
import java.util.concurrent.ThreadLocalRandom;
int num4 = ThreadLocalRandom.current().nextInt(100);
```

## 五、日期时间 API

### 5.1 JDK 8 之前的日期API（不推荐）

```java
import java.util.Date;
import java.text.SimpleDateFormat;

// Date类
Date date = new Date();
System.out.println(date);

// SimpleDateFormat
SimpleDateFormat sdf = new SimpleDateFormat("yyyy-MM-dd HH:mm:ss");
String dateStr = sdf.format(date);
System.out.println(dateStr);

// 解析
Date parsedDate = sdf.parse("2024-01-01 12:00:00");
```

**缺点：**
- 线程不安全
- API设计混乱
- 时区处理复杂

### 5.2 JDK 8 新的日期API（推荐）

#### LocalDate（日期）

```java
import java.time.LocalDate;

// 当前日期
LocalDate today = LocalDate.now();

// 指定日期
LocalDate birthday = LocalDate.of(1990, 1, 1);
LocalDate birthday2 = LocalDate.parse("1990-01-01");

// 获取年月日
int year = today.getYear();
int month = today.getMonthValue();
int day = today.getDayOfMonth();

// 加减操作
LocalDate tomorrow = today.plusDays(1);
LocalDate lastWeek = today.minusWeeks(1);

// 比较
boolean isBefore = birthday.isBefore(today);
boolean isAfter = birthday.isAfter(today);
```

#### LocalTime（时间）

```java
import java.time.LocalTime;

// 当前时间
LocalTime now = LocalTime.now();

// 指定时间
LocalTime time = LocalTime.of(14, 30, 0);

// 获取时分秒
int hour = now.getHour();
int minute = now.getMinute();
int second = now.getSecond();
```

#### LocalDateTime（日期时间）

```java
import java.time.LocalDateTime;

// 当前日期时间
LocalDateTime now = LocalDateTime.now();

// 指定日期时间
LocalDateTime dateTime = LocalDateTime.of(2024, 1, 1, 12, 0, 0);

// 格式化
import java.time.format.DateTimeFormatter;
DateTimeFormatter formatter = DateTimeFormatter.ofPattern("yyyy-MM-dd HH:mm:ss");
String formatted = now.format(formatter);

// 解析
LocalDateTime parsed = LocalDateTime.parse("2024-01-01 12:00:00", formatter);
```

#### Duration 和 Period

```java
import java.time.Duration;
import java.time.Period;
import java.time.LocalDateTime;

LocalDateTime start = LocalDateTime.now();
LocalDateTime end = start.plusHours(2);

// Duration：计算时间差（小时、分钟、秒）
Duration duration = Duration.between(start, end);
long hours = duration.toHours();        // 2
long minutes = duration.toMinutes();    // 120

// Period：计算日期差（年、月、日）
LocalDate date1 = LocalDate.of(2020, 1, 1);
LocalDate date2 = LocalDate.of(2024, 1, 1);
Period period = Period.between(date1, date2);
int years = period.getYears();    // 4
int months = period.getMonths();  // 0
int days = period.getDays();      // 0
```

#### Instant（时间戳）

```java
import java.time.Instant;

// 当前时间戳
Instant now = Instant.now();

// 毫秒数
long millis = now.toEpochMilli();

// 从毫秒创建
Instant instant = Instant.ofEpochMilli(System.currentTimeMillis());
```

### 5.3 新旧API转换

```java
import java.time.Instant;
import java.time.ZoneId;
import java.util.Date;

// Date → LocalDateTime
Date date = new Date();
LocalDateTime ldt = date.toInstant()
    .atZone(ZoneId.systemDefault())
    .toLocalDateTime();

// LocalDateTime → Date
LocalDateTime localDateTime = LocalDateTime.now();
Date newDate = Date.from(localDateTime.atZone(ZoneId.systemDefault()).toInstant());
```

## 八、Objects 工具类

### 8.1 常用方法

```java
import java.util.Objects;

// 判断是否为null
Objects.isNull(obj);      // obj == null
Objects.nonNull(obj);     // obj != null

// 要求不为null，否则抛异常
Objects.requireNonNull(obj);
Objects.requireNonNull(obj, "对象不能为null");

// 比较两个对象
Objects.equals(obj1, obj2);  // 安全比较，避免NullPointerException

// 获取哈希码
int hash = Objects.hash(field1, field2, field3);

// toString
String str = Objects.toString(obj, "default");
```

### 8.2 实际应用

```java
public class Person {
    private String name;
    private Integer age;
    
    @Override
    public boolean equals(Object o) {
        if (this == o) return true;
        if (o == null || getClass() != o.getClass()) return false;
        Person person = (Person) o;
        return Objects.equals(name, person.name) &&
               Objects.equals(age, person.age);
    }
    
    @Override
    public int hashCode() {
        return Objects.hash(name, age);
    }
}
```

## 十一、BigDecimal（高精度计算）

### 11.1 为什么需要 BigDecimal？

float 和 double 存在精度问题：

```java
System.out.println(0.1 + 0.2);  // 0.30000000000000004
```

### 11.2 使用方法

```java
import java.math.BigDecimal;

// 创建（推荐使用String构造）
BigDecimal num1 = new BigDecimal("0.1");
BigDecimal num2 = new BigDecimal("0.2");

// 加法
BigDecimal sum = num1.add(num2);  // 0.3

// 减法
BigDecimal diff = num2.subtract(num1);  // 0.1

// 乘法
BigDecimal product = num1.multiply(num2);  // 0.02

// 除法
BigDecimal quotient = num2.divide(num1);  // 2

// 除法（指定精度和舍入模式）
BigDecimal result = new BigDecimal("10").divide(
    new BigDecimal("3"), 
    2, 
    BigDecimal.ROUND_HALF_UP
);  // 3.33

// 比较
int cmp = num1.compareTo(num2);  // -1（小于）

// 转换为基本类型
double d = num1.doubleValue();
int i = num1.intValue();
```

### 11.3 注意事项

```java
// 错误！不要用double构造
BigDecimal wrong = new BigDecimal(0.1);  // 精度问题

// 正确！用String构造
BigDecimal right = new BigDecimal("0.1");

// 或者使用valueOf
BigDecimal alsoRight = BigDecimal.valueOf(0.1);
```

## 十二、Optional（避免空指针）

### 12.1 创建 Optional

```java
import java.util.Optional;

// 创建
Optional<String> opt1 = Optional.of("Hello");       // 不能为null
Optional<String> opt2 = Optional.ofNullable(null);  // 可以为null
Optional<String> opt3 = Optional.empty();           // 空Optional
```

### 12.2 常用方法

```java
Optional<String> opt = Optional.ofNullable(getName());

// 判断是否有值
boolean present = opt.isPresent();

// 获取值（如果为空抛异常）
String value = opt.get();

// 如果有值则执行
opt.ifPresent(name -> System.out.println(name));

// 提供默认值
String name = opt.orElse("Unknown");

// 提供默认值（Supplier）
String name2 = opt.orElseGet(() -> "Default");

// 为空时抛异常
String name3 = opt.orElseThrow(() -> new RuntimeException("值为空"));

// 过滤
Optional<String> filtered = opt.filter(name -> name.length() > 3);

// 转换
Optional<Integer> length = opt.map(String::length);
```

### 12.3 实际应用

```java
// 传统方式
User user = getUser();
if (user != null) {
    Address address = user.getAddress();
    if (address != null) {
        String city = address.getCity();
        if (city != null) {
            System.out.println(city);
        }
    }
}

// Optional方式
String city = Optional.ofNullable(getUser())
    .map(User::getAddress)
    .map(Address::getCity)
    .orElse("Unknown");
```
